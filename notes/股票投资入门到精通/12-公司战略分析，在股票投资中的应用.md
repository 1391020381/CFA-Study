# 公司战略的定义
    - 确定企业的基本长期目标，选择行动途径和为实现这些目标，进行资源分配活动。
# 公司战略回答三个问题
    - 公司的目标和愿景是什么
    - 公司在行业中 竞争优势是是什么
    - 基于公司的目标和愿景，执行路径和行动计划是什么？
# 股票投资为什么要学习战略分析
1. 战略决定公司天花板

案例：比亚迪\ 垂直整合战略实现成本领先，后通过开放电池业务、共享技术平台，从单一车企转型为新能源核心供应商，市值上限显著提升。\ 战略选择直接影响公司规模与价值。


2. 战略是未来盈利的蓝图

案例：宁德时代\ 技术领先战略（如麒麟电池）+客户绑定战略（与特斯拉等车企合作建厂），确保产品溢价与市场份额，支撑未来业绩持续增长。\ 战略规划比短期财报更能反映长期价值。


3. 战略执行反映管理水平

案例：东方雨虹公司制定渠道变革、多元化等高质量发展战略，但执行中为短期业绩激进赊账，导致房地产调整期坏账激增。\ 战略执行偏差暴露管理风险，需警惕 “口号式战略”。


4. 识别风险与机会

案例：格力电器 vs 阳光电源
格力电器：依赖空调主业，多元化手机、新能源汽车失败，业绩停滞，估值下降。
阳光电源：围绕光伏、储能、氢能的协同战略，推动业绩独立增长，提前布局行业爆发。\ 战略清晰与否决定企业能否穿越周期。




5. 战略提供决策锚点

案例：贵州茅台坚守高端品牌定位，维护稀缺性，即便短期业绩波动（如禁酒令、消费低迷），战略定力支撑长期价值。\ 战略理解帮助投资者抵御短期噪音，避免追涨杀跌。


核心结论
战略分析是股票投资的 “导航系统”，通过理解公司长期目标、竞争优势与执行路径，能更准确判断价值与风险，实现理性决策与长期收益。
