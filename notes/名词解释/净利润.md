# 净利润（Net Income）

## 定义

**净利润** = 公司**最终**赚到的、属于股东的利润。

简单来说：**扣除所有成本、费用、税金后，真正落到股东口袋里的钱 —— 利润表的"最后一行"。**

---

## 核心公式

```
净利润 = 利润总额 - 所得税费用
       = 营业利润 + 营业外收入 - 营业外支出 - 所得税
```

---

## 净利润的意义

### 1. 股东真正能分到的钱

```
净利润 = 可以分红的钱
```

| 净利润情况 | 含义 |
|-----------|------|
| 净利润 > 0 | 赚钱了，可以分红 ✅ |
| 净利润 < 0 | 亏钱了，不能分红 ❌ |

**但注意**：净利润 ≠ 现金！

---

### 2. 进入留存收益

```
净利润 → 留存收益 → 所有者权益
```

| 分红情况 | 留存收益变化 |
|----------|--------------|
| 全部分红 | 不变 |
| 分一半 | 增加一半 |
| 不分红 | 增加全部 |

**茅台**（2023年估算）：
- 净利润700亿
- 分红350亿
- 留存收益增加350亿

---

### 3. 每股收益（EPS）的基础

```
每股收益（EPS）= 净利润 ÷ 总股本
```

**茅台**（2023年估算）：
- 净利润700亿
- 总股本12.6亿股
- EPS = 700 ÷ 12.6 ≈ **55.6元/股**

---

## 净利润的构成

### 利润表完整流程

```
营业收入 750亿
    │
    ├─ 减：营业成本 -70亿
    │
    ├─ = 毛利 680亿
    │
    ├─ 减：期间费用 -100亿
    │
    ├─ = 营业利润 580亿
    │
    ├─ 加：营业外收入 0
    ├─ 减：营业外支出 0
    │
    ├─ = 利润总额 580亿
    │
    ├─ 减：所得税费用 -145亿
    │
    └─ = 净利润 435亿 ✅
```

**茅台2023年（估算）**：
- 净利润约**435亿元**
- 净利率约**58%**（435 ÷ 750）

---

## 净利率

### 净利率计算

```
净利率 = 净利润 ÷ 营业收入 × 100%
```

| 净利率水平 | 评价 |
|-----------|------|
| > 30% | 极高 ✅（茅台58%） |
| 20%-30% | 很好 |
| 10%-20% | 良好 |
| 5%-10% | 一般 |
| < 5% | 很低 ⚠️ |
| < 0% | 亏损 ❌ |

---

### 行业净利率对比

| 行业 | 净利率 | 原因 |
|------|--------|------|
| **白酒高端** | 40%-60% | 定价权强、费用率低 |
| **软件/SaaS** | 20%-40% | 边际成本低 |
| **奢侈品** | 15%-25% | 品牌溢价 |
| **医药** | 10%-20% | 研发费用高 |
| **零售** | 2%-5% | 竞争激烈、成本高 |
| **航空** | 3%-10% | 成本高、周期性 |
| **钢铁** | 2%-8% | 重资产、周期波动 |

---

## 茅台的净利率

### 茅台净利率（估算）

| 年份 | 收入（亿） | 净利润（亿） | 净利率 |
|------|-----------|-------------|--------|
| 2018 | 600 | 300 | 50% |
| 2019 | 650 | 340 | 52% |
| 2020 | 700 | 370 | 53% |
| 2021 | 750 | 400 | 53% |
| 2022 | 800 | 450 | 56% |
| 2023 | 750 | 435 | 58% |

**分析**：
- 净利率持续上升
- 从50% → 58%
- 说明盈利能力在增强

---

### 为什么茅台净利率这么高？

| 原因 | 说明 |
|------|------|
| **毛利率极高** | 91%，产品本身赚钱 |
| **期间费用率低** | 13%，运营效率高 |
| **财务费用为负** | 利息收入大于支出 |
| **所得税正常** | 25%，正常水平 |

**公式**：
```
净利率 = 毛利率 - 期间费用率 - 所得税率
       = 91% - 13% - 20%
       = 58%
```

---

## 净利润 vs 现金流

### 关键区别：净利润 ≠ 现金

| 项目 | 净利润 | 经营现金流 |
|------|--------|-----------|
| **原则** | 权责发生制 | 收付实现制 |
| **时间** | 交易发生时确认 | 实际收付现金时 |
| **可靠性** | 可能被操纵 | 较难操纵 |

---

### 例子：赊销

**场景**：茅台12月卖出10亿酒，经销商3个月后付款

| 项目 | 12月 | 3月 |
|------|------|------|
| 净利润 | ✅ +10亿 | 0 |
| 经营现金流 | 0 | ✅ +10亿 |

**关键理解**：
- 12月有净利润，但没有现金
- 3月有现金，但没有当期净利润
- 这就是"利润 ≠ 现金"

---

### 净利润现金比率

```
净利润现金比率 = 经营现金流 ÷ 净利润
```

| 比率 | 含义 |
|------|------|
| > 1 | 利润质量很好 ✅ |
| ≈ 1 | 利润质量正常 |
| < 1 | 利润质量一般 ⚠️ |
| < 0 | 可能有问题 ❌ |

**茅台**：净利润现金比率通常 > 1，说明利润质量极好！

---

## 净利润的"陷阱"

### 陷阱1：净利润高但没现金

**原因**：大量赊销

**例子**：A公司

| 项目 | 金额 |
|------|------|
| 净利润 | 10亿 |
| 经营现金流 | -5亿 |
| 应收账款 | 增加15亿 |

**问题**：
- 利润表显示赚钱
- 但没有现金流入
- 应收账款可能变成坏账

---

### 陷阱2：一次性收益虚增净利润

**原因**：卖资产、政府补贴

**例子**：B公司

| 项目 | 金额 |
|------|------|
| 营业利润 | 5亿（主业） |
| 卖厂房收益 | 10亿（一次性） |
| 净利润 | 15亿（虚高） |

**问题**：
- 净利润看起来很好
- 但主业只赚5亿
- 卖厂房是不可持续的

---

### 陷阱3：净利润被操纵

**手段**：
- 少计提折旧
- 少计提减值
- 提前确认收入
- 延迟确认费用

**如何识别**：
- 看经营现金流与净利润是否匹配
- 看应收账款是否异常增长
- 看存货是否异常增长

---

## 如何用净利润分析公司

### 第一步：看净利率水平

```
净利率 > 20% = 优秀 ✅
净利率 10%-20% = 良好
净利率 < 10% = 一般 ⚠️
净利率 < 0% = 亏损 ❌
```

---

### 第二步：看净利润趋势

```
连续增长 = 盈利能力提升 ✅
连续下降 = 盈利能力恶化 ⚠️
波动大 = 行业周期性
```

---

### 第三步：看净利润 vs 经营现金流

| 情况 | 含义 |
|------|------|
| 经营现金流 > 净利润 | 利润质量很好 ✅ |
| 经营现金流 ≈ 净利润 | 利润质量正常 |
| 经营现金流 < 净利润 | 利润质量一般 ⚠️ |
| 经营现金流 << 0 | 可能有风险 ❌ |

---

### 第四步：看净利润来源

| 情况 | 含义 |
|------|------|
| 净利润 ≈ 营业利润 | 盈利来自主业 ✅ |
| 净利润 >> 营业利润 | 盈利来自副业 ⚠️ |
| 净利润 << 营业利润 | 税收或营业外支出大 ⚠️ |

<summary>点击查看答案</summary>

A公司盈利能力强。

</details>
