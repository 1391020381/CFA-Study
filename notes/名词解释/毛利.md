# 毛利（Gross Profit）

## 定义

**毛利** = 营业收入 - 营业成本

简单来说：**产品卖出去，扣除直接成本后剩下的钱 —— 还没扣除各项费用之前的"第一层利润"。**

---

## 核心公式

```
毛利 = 营业收入 - 营业成本
```

---

## 毛利的意义

### 1. 产品盈利能力

```
毛利 > 0 → 产品赚钱
毛利 < 0 → 产品亏本
```

| 毛利情况 | 含义 |
|----------|------|
| 毛利高 | 产品有竞争力 ✅ |
| 毛利低 | 价格战激烈或成本高 |
| 毛利为负 | 卖得越多亏得越多 ❌ |

---

### 2. 覆盖期间费用的能力

```
毛利 - 期间费用 = 营业利润
```

**茅台例子**（估算）：
- 毛利：680亿
- 期间费用：150亿
- 营业利润：530亿

**关键理解**：毛利必须足够高，才能覆盖销售、管理、研发等费用，还有剩余。

---

## 毛利 vs 净利润

| 项目 | 公式 | 意义 |
|------|------|------|
| **毛利** | 收入 - 成本 | 产品本身赚多少 |
| **净利润** | 毛利 - 费用 - 税 | 最终赚多少 |

**从毛利到净利润的路径**：

```
毛利（680亿）
    │
    ├─ 减：销售费用（50亿）
    ├─ 减：管理费用（80亿）
    ├─ 减：研发费用（10亿）
    ├─ 减：财务费用（-10亿，即收入）
    │
    ├─ = 营业利润（530亿）
    │
    ├─ 加：营业外收支（0）
    ├─ 减：所得税（130亿）
    │
    └─ = 净利润（400亿）
```

## 茅台的毛利（估算）

| 项目 | 金额 |
|------|------|
| 营业收入 | 750亿 |
| 营业成本 | 70亿 |
| **毛利** | **680亿** |
| **毛利率** | **91%** |

**解读**：
- 每卖100元茅台酒，毛利91元
- 这是极其恐怖的盈利能力
- 只有极少数公司能达到

---

## 毛利的对比分析

### 行业对比

| 行业 | 典型毛利率 | 原因 |
|------|-----------|------|
| **白酒** | 70%-90%+ | 品牌溢价、定价权 |
| **软件** | 80%-95% | 边际成本几乎为0 |
| **奢侈品** | 70%-85% | 品牌价值 |
| **餐饮** | 50%-70% | 食材成本占比高 |
| **零售** | 10%-30% | 竞争激烈、成本高 |
| **钢铁** | 5%-15% | 重资产、周期性 |

---

### 公司对比（白酒行业）

| 公司 | 毛利率 | 特点 |
|------|--------|------|
| **茅台** | 91% | 最高端，定价权最强 |
| **五粮液** | 75% | 高端，但略低于茅台 |
| **泸州老窖** | 70% | 次高端 |
| **低端白酒** | 40%-60% | 价格战激烈 |

**茅台的优势**：
- 品牌最强，定价权最强
- 毛利率持续领先

---

## 毛利率分析

### 毛利率计算

```
毛利率 = 毛利 ÷ 营业收入 × 100%
```

| 毛利率水平 | 评价 |
|-----------|------|
| > 70% | 极高（有定价权） ✅ |
| 50%-70% | 很好 |
| 30%-50% | 一般 |
| 20%-30% | 较低 |
| < 20% | 很低 ⚠️ |

---

### 毛利率变化分析

| 变化 | 可能原因 |
|------|----------|
| **毛利率上升** ✅ | 产品涨价、成本下降、产品结构优化 |
| **毛利率下降** ⚠️ | 价格战、成本上升、低毛利产品占比增加 |

**茅台历史毛利率**（估算）：
- 2018年：90%
- 2020年：91%
- 2023年：91%
- 稳定在极高水平

---

## 毛利与定价权

### 定价权的体现

```
定价权 = 能够提价而不影响销量的能力
```

| 公司 | 定价权 | 毛利率表现 |
|------|--------|-----------|
| **茅台** | 极强 ✅ | 持续90%+ |
| **苹果** | 很强 ✅ | 持续40%+ |
| **普通制造业** | 弱 ❌ | 随成本波动 |

**茅台的定价权**：
- 2010年：500元/瓶
- 2020年：1500元/瓶
- 2023年：2500元/瓶（市场价）
- 价格涨了5倍，毛利率反而上升！

---

## 毛利的"陷阱"

### 陷阱1：虚假高毛利

**手段**：少结转成本

**例子**：
- 应该结转100的成本，只结转80
- 毛利虚高20

**如何识别**：
- 看存货周转率是否异常下降
- 看毛利率是否显著高于同行
- 看现金流与利润是否匹配

---

### 陷阱2：不可持续的高毛利

**原因**：一次性因素

| 一次性因素 | 例子 |
|-----------|------|
| 原材料价格短期下降 | 猪肉价格下跌，肉制品毛利上升 |
| 一次性补贴 | 政府补贴提高毛利 |
| 暂时性短缺 | 供不应求，价格上涨 |

**判断**：看长期毛利率是否稳定

---

## 如何提高毛利

### 1. 提高价格（提价）

| 适用情况 | 风险 |
|----------|------|
| 有定价权的公司 ✅ | 需求弹性小 |
| 产品差异化 | 可能影响销量 |

**茅台**：可以不断提价，销量不减

---

### 2. 降低成本

| 方法 | 效果 |
|------|------|
| 规模效应 | 单位成本下降 |
| 供应链优化 | 降低采购成本 |
| 技术升级 | 提高效率 |

---

### 3. 优化产品结构

```
高毛利产品占比 ↑ → 整体毛利 ↑
```

**例子**：
- 茅台增加高价的"飞天茅台"占比
- 减少低价的系列酒占比

## 学习检查

**问题1**：A公司收入100亿，成本60亿；B公司收入100亿，成本40亿。谁的盈利能力强？

<details>
<summary>点击查看答案</summary>

B公司盈利能力更强。
- A公司毛利40亿，毛利率40%
- B公司毛利60亿，毛利率60%
- B公司产品竞争力更强

</details>

---

**问题2**：茅台为什么毛利率高达90%+？

<details>
<summary>点击查看答案</summary>

- 品牌极强，有定价权
- 可以定高价
- 生产成本相对固定
- 原材料成本占比低
- 供不应求

</details>

---

**问题3**：如果一家公司毛利率突然从30%上升到50%，你应该警惕什么？

<details>
<summary>点击查看答案</summary>

- 可能是少结转成本（财务造假）
- 可能是一次性因素（不可持续）
- 要对比同行毛利率
- 要检查现金流是否匹配

</details>

```
毛利 = 收入 - 成本

警惕：突然变化、显著高于同行
```
