# 流动比率（Current Ratio）

## 定义

**流动比率** = 流动资产 ÷ 流动负债。

简单来说：**公司短期内（一年内）能变现的资产，能否覆盖短期内要还的债 —— 衡量短期偿债能力的核心指标。**

---

## 核心公式

```
流动比率 = 流动资产 ÷ 流动负债
```

**茅台例子**（2023年估算）：

| 项目 | 金额（亿元） |
|------|-------------|
| 流动资产 | 1800 |
| 流动负债 | 300 |
| **流动比率** | **6.0** |

**含义**：茅台的流动资产是流动负债的6倍，短期偿债能力极强！

---

## 流动比率的意义

### 1. 衡量短期偿债能力

```
流动比率 > 1 → 能还债 ✅
流动比率 < 1 → 可能还不上债 ❌
```

**理解**：
- 流动资产：一年内能变现的资产（现金、存货、应收账款）
- 流动负债：一年内要还的债（应付账款、短期借款）
- 流动比率 = 这两者能否匹配

---

### 2. 安全边际

```
流动比率 = 还债的"安全垫"
```

| 公司A | 公司B |
|--------|--------|
| 流动资产100万 | 流动资产100万 |
| 流动负债90万 | 流动负债30万 |
| 流动比率1.1 | 流动比率3.3 |
| **勉强够还** | **很安全** ✅ |

**理解**：
- 公司A：流动资产跌10%，就还不上了
- 公司B：流动资产跌70%，还能还上

---

## 流动比率的评判标准

### 通用标准

| 流动比率 | 偿债能力 | 评价 |
|----------|----------|------|
| **> 3** | 极强 ✅ | 非常安全 |
| **2-3** | 很强 ✅ | 健康 |
| **1.5-2** | 较强 ✅ | 正常 |
| **1-1.5** | 一般 ⚠️ | 要注意 |
| **< 1** | 较弱 ❌ | 有压力 |

**茅台**：流动比率约6（极强）

---

### 行业差异

| 行业 | 典型流动比率 | 原因 |
|------|-------------|------|
| **白酒** | 3-6 | 现金多、周转快 |
| **零售** | 1.5-2.5 | 存货多但周转快 |
| **制造业** | 1.5-2 | 正常水平 |
| **建筑** | 1-1.5 | 应收账款多 |
| **公用事业** | 1-1.5 | 现金流稳定 |

---

## 茅台的流动比率分析

### 茅台流动比率（估算）

| 年份 | 流动资产（亿） | 流动负债（亿） | 流动比率 |
|------|---------------|---------------|----------|
| 2018 | 1500 | 300 | 5.0 |
| 2019 | 1600 | 320 | 5.0 |
| 2020 | 1700 | 330 | 5.2 |
| 2021 | 1750 | 340 | 5.1 |
| 2022 | 1800 | 350 | 5.1 |
| 2023 | 1800 | 300 | 6.0 |

**分析**：
- 流动比率持续在**5-6**之间
- 极高的短期偿债能力
- 不可能有资金链断裂风险

---

### 为什么茅台流动比率这么高？

| 原因 | 说明 |
|------|------|
| **现金极多** | 货币资金600+亿 |
| **应收账款极少** | 基本不赊销 |
| **存货是优质资产** | 存酒越陈越值钱 |
| **流动负债少** | 预收账款多（不需要还钱，只要发货） |

---

## 流动比率 vs 速动比率

### 速动比率（更严格）

```
速动比率 = (流动资产 - 存货) ÷ 流动负债
```

**为什么扣除存货？**
- 存货可能卖不出去
- 存货可能贬值
- 存货变现慢

**茅台**（估算）：

| 项目 | 金额（亿） |
|------|-------------|
| 流动资产 | 1800 |
| 存货 | 400 |
| 速动资产 | 1400 |
| 流动负债 | 300 |
| **流动比率** | **6.0** |
| **速动比率** | **4.7** |

**分析**：
- 即使扣除存货，速动比率仍有4.7
- 短期偿债能力依然极强

---

## 流动比率的"陷阱"

### 陷阱1：流动比率虚高

**原因**：存货积压

**例子**：某公司

| 项目 | 金额（亿） |
|------|-----------|
| 流动资产 | 200 |
| - 存货（积压） | 150 |
| - 速动资产 | 50 |
| 流动负债 | 100 |
| 流动比率 | 2.0（看起来不错） |
| 速动比率 | 0.5（实际很差） |

**问题**：
- 流动比率2看起来不错
- 但存货占150亿，可能卖不出去
- 速动比率只有0.5，实际偿债能力差

---

### 陷阱2：流动比率低但现金流好

**例子**：某超市

| 项目 | 金额 |
|------|------|
| 流动资产 | 50 |
| 流动负债 | 60 |
| 流动比率 | 0.83（< 1） |
| 日均现金流 | +5万 |

**分析**：
- 流动比率 < 1
- 但每天都有现金流入
- 实际偿债能力不差

**结论**：流动比率要结合现金流看！

---

## 如何用流动比率分析公司

### 第一步：看流动比率水平

```
> 2 ✅ = 很安全
1.5-2 = 正常
1-1.5 ⚠️ = 要注意
< 1 ❌ = 有压力
```

---

### 第二步：看趋势

```
持续上升 ✅ = 偿债能力增强
持续下降 ⚠️ = 偿债能力恶化
稳定 = 正常
```

---

### 第三步：看速动比率

```
速动比率 > 1.5 ✅ = 很安全
速动比率 < 1 ⚠️ = 要注意
```

---

### 第四步：看流动资产质量

| 资产类型 | 质量 |
|----------|------|
| 现金、理财 | 极好 ✅ |
| 应收账款 | 要看账龄 |
| 存货 | 要看周转率 |

---

### 第五步：结合现金流

```
流动比率好 + 经营现金流好 ✅ = 真正安全
流动比率好 + 经营现金流差 ⚠️ = 可能有问题
```

---

## 流动比率与资产负债率

### 短期 vs 长期偿债能力

| 指标 | 衡量 | 期限 |
|------|------|------|
| **流动比率** | 短期偿债能力 | 1年内 |
| **资产负债率** | 长期偿债能力 | 全部 |

**茅台**：
- 流动比率：6.0（短期极强）✅
- 资产负债率：18%（长期极强）✅
- 双重安全！

**问题2**：B公司流动资产100万，流动负债120万。流动比率是多少？这说明什么？

<details>
<summary>点击查看答案</summary>

流动比率 = 100 ÷ 120 = 0.83

- 短期偿债能力不足

<details>
<summary>点击查看答案</summary>

- 流动比率高是因为存货多
- 扣除存货后，速动比率只有0.8
- 存货可能卖不出去或贬值
- 要看存货周转率

</details>
