# 经营活动现金流（Operating Cash Flow）

## 定义

**经营活动现金流** = 公司**主营业务**产生的现金收支。

简单来说：**公司靠做生意真正收到的现金和花的现金 —— 主业的"造血能力"。**

---

## 核心公式

```
经营活动净现金流 = 经营活动现金流入 - 经营活动现金流出
```

---

## 经营活动包括什么？

### 现金流入（钱进来了）

| 项目 | 说明 | 例子 |
|------|------|------|
| **销售商品收到的现金** | 卖货实际收到的钱 | 茅台卖酒收到的钱 |
| **收到的税费返还** | 政府退税 | 出口退税 |
| **收到其他与经营有关的现金** | 其他经营收入 | 罚款收入、利息收入 |

**茅台**：主要流入是"销售商品收到的现金"

---

### 现金流出（钱花出去了）

| 项目 | 说明 | 例子 |
|------|------|------|
| **购买商品支付的现金** | 买原材料 | 买粮食、包装 |
| **支付职工现金** | 发工资 | 酿酒师傅工资 |
| **支付税费** | 交税 | 增值税、所得税 |
| **支付其他经营费用** | 办公、差旅等 | 水电费、办公费 |

**茅台**：主要是买原料、发工资、交税

---

## 经营活动现金流 vs 净利润

### 最核心的区别

| 项目 | 净利润 | 经营现金流 |
|------|--------|-----------|
| **原则** | 权责发生制 | 收付实现制 |
| **时间** | 交易发生时确认 | 实际收付现金时 |
| **可靠性** | 可能被操纵 | 较难操纵 |

---

### 例子：赊销场景

**场景**：茅台12月卖出10亿酒，经销商3个月后付款

| 项目 | 12月 | 3月 |
|------|------|------|
| **净利润** | ✅ +10亿 | 0 |
| **经营现金流** | 0 | ✅ +10亿 |

**关键理解**：
- 12月：有利润，没现金
- 3月：有现金，没当期利润
- 这就是"利润 ≠ 现金"

---

## 净利润现金比率

### 核心公式

```
净利润现金比率 = 经营活动现金流 ÷ 净利润
```

### 如何判断

| 比率 | 含义 |
|------|------|
| **> 1** | 利润质量很好 ✅ |
| **≈ 1** | 利润质量正常 |
| **0.5-1** | 利润质量一般 ⚠️ |
| **< 0.5** | 利润质量较差 ⚠️ |
| **< 0** | 主业不赚钱 ❌ |

---

### 为什么会 > 1？

**茅台例子**（2023年估算）：

| 项目 | 金额（亿） |
|------|-----------|
| 净利润 | 400 |
| 经营现金流 | 445 |
| 净利润现金比率 | 1.11 ✅ |

**为什么现金流 > 利润？**

| 原因 | 说明 |
|------|------|
| **折旧** | 净利润扣了折旧，但没现金流出 |
| **摊销** | 同上 |
| **预收账款** | 收到现金但没确认收入 |

**茅台**：
- 折旧+摊销约50亿
- 预收账款增加
- 所以现金流 > 利润

---

### 为什么会 < 1？

**常见原因**：

| 原因 | 说明 |
|------|------|
| **应收账款增加** | 赊销多，没收到现金 |
| **存货增加** | 买了大量原料，占用现金 |
| **预付账款增加** | 预先付钱给供应商 |

**危险信号** ⚠️：
- 净利润很高，但经营现金流很低或为负
- 说明"纸面富贵"，利润质量差

---

## 茅台的经营现金流

### 茅台经营现金流（估算）

| 年份 | 净利润（亿） | 经营现金流（亿） | 比率 |
|------|-------------|-----------------|------|
| 2018 | 300 | 350 | 1.17 |
| 2019 | 340 | 400 | 1.18 |
| 2020 | 370 | 420 | 1.14 |
| 2021 | 400 | 450 | 1.13 |
| 2022 | 450 | 500 | 1.11 |
| 2023 | 400 | 445 | 1.11 |

**分析**：
- 净利润现金比率持续 > 1
- 说明利润质量极好
- 茅台是"现金奶牛"

---

### 为什么茅台经营现金流这么好？

| 原因 | 说明 |
|------|------|
| **先款后货** | 经销商预付款 |
| **应收账款极少** | 不需要赊销 |
| **存货是资产** | 存酒越陈越值钱 |
| **折旧摊销大** | 减少净利润但没现金流出 |

---

## 经营现金流的"危险信号"

### 信号1：经营现金流长期为负

```
经营现金流 < 0 → 主业不赚钱 ❌
```

**原因**：
- 产品卖不出去
- 成本太高
- 赊销太多

**这类公司**：
- 创业公司（正常）
- 衰退公司（危险）

---

### 信号2：净利润高但现金流低

```
净利润 > 0，经营现金流 ≈ 0 或 < 0
```

**例子**：A公司

| 项目 | 金额 |
|------|------|
| 净利润 | 10亿 |
| 经营现金流 | 2亿 |
| 应收账款 | +8亿 |

**问题**：
- 赊销严重
- 利润质量差
- 可能坏账风险

---

### 信号3：持续依赖融资维持

```
经营现金流为负，靠投资/筹资现金流补充
```

**例子**：B公司

| 项目 | 金额（亿） |
|------|-----------|
| 经营现金流 | -20 |
| 投资现金流 | -10 |
| 筹资现金流 | +30 |

**问题**：
- 主业造血能力差
- 需要不断融资"输血"
- 资金链断裂风险高

---

## 如何用经营现金流分析公司

### 第一步：看经营现金流是否为正

```
持续为正 ✅ = 主业赚钱
持续为负 ❌ = 主业亏钱
```

---

### 第二步：看净利润现金比率

```
> 1 ✅ = 利润质量很好
0.5-1 ⚠️ = 利润质量一般
< 0.5 ❌ = 利润质量差
```

---

### 第三步：看趋势

```
持续增长 ✅ = 造血能力增强
持续下降 ⚠️ = 造血能力恶化
波动大 = 行业周期性
```

---

### 第四步：对比净利润和经营现金流

| 情况 | 含义 |
|------|------|
| 经营现金流 ≈ 净利润 | 正常 ✅ |
| 经营现金流 > 净利润 | 很好 ✅ |
| 经营现金流 < 净利润 | 要警惕 ⚠️ |
| 经营现金流 << 净利润 | 危险 ❌ |

## 学习检查

**问题1**：A公司净利润10亿，经营现金流5亿。这说明了什么？

<details>
<summary>点击查看答案</summary>

利润质量一般 ⚠️
- 净利润现金比率 = 5 ÷ 10 = 0.5
- 说明可能赊销较多
- 或者存货占用大量现金

</details>

---

**问题2**：为什么茅台的净利润现金比率 > 1？

<details>
<summary>点击查看答案</summary>

- 折旧摊销减少净利润但没现金流出
- 预收账款增加（收到现金但未确认收入）
- 应收账款很少（基本不赊销）
- 所以现金流 > 利润

</details>

---

**问题3**：B公司连续3年经营现金流为负，但净利润为正。你应该警惕什么？

<details>
<summary>点击查看答案</summary>

危险信号 ❌
- 主业不产生现金
- 利润可能是"纸面富贵"
- 大量赊销或存货积压
- 资金链断裂风险高

</details>
