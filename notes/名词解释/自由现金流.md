# 自由现金流（Free Cash Flow）

## 定义

**自由现金流** = 公司**可以自由支配**的现金。

简单来说：**经营现金流减去必须的投入后，剩下的钱 —— 可以分红、回购、还债的"真金白银"。**

---

## 核心公式

```
自由现金流 = 经营活动现金流 - 资本开支
```

---

## 为什么叫"自由"现金流？

### "自由"的意思

| 约束 | 说明 |
|------|------|
| **资本开支** | 必须花的钱（买设备、建厂房） |
| **自由现金流** | 扣除必须开支后，"自由"支配的钱 |

**可以用来**：
- 分红给股东 ✅
- 回购股票 ✅
- 还债 ✅
- 收购其他公司 ✅
- 存银行 ✅

**不能用来的**（已扣除）：
- 必须的设备更新
- 必须的厂房维护

---

## 自由现金流的计算

### 方法1：直接法

```
自由现金流 = 经营活动现金流 - 资本开支
```

**茅台例子**（2023年估算）：

| 项目 | 金额（亿元） |
|------|-------------|
| 经营活动现金流 | 445 |
| 资本开支 | 50 |
| **自由现金流** | **395** ✅ |

**分析**：
- 茅台有395亿"自由"现金
- 可以分红、回购、投资
- 这是极高的自由现金流！

---

### 方法2：间接法

```
自由现金流 = 净利润 + 折旧摊销 - 营运资本增加 - 资本开支
```

**为什么要加回折旧摊销？**
- 折旧减少了净利润
- 但没有现金流出
- 所以要加回来

---

## 自由现金流 vs 净利润

### 关键区别

| 项目 | 净利润 | 自由现金流 |
|------|--------|-----------|
| **原则** | 权责发生制 | 收付实现制 |
| **真实性** | 可以被操纵 | 较难操纵 |
| **意义** | 会计利润 | 真实现金 |

---

### 例子：A公司

| 项目 | 金额（亿） |
|------|-----------|
| 净利润 | 100 |
| + 折旧摊销 | 20 |
| - 营运资本增加 | 30 |
| - 资本开支 | 40 |
| **自由现金流** | **50** |

**分析**：
- 净利润100亿
- 自由现金流只有50亿
- 说明一半利润是"纸面富贵"

---

## 自由现金流的意义

### 1. 真实的盈利能力

```
自由现金流 > 0 → 公司真正赚钱 ✅
自由现金流 < 0 → 公司在烧钱 ⚠️
```

**茅台**：自由现金流395亿，真正的现金奶牛！

---

### 2. 分红能力

```
自由现金流 > 分红 → 分红可持续 ✅
自由现金流 < 分红 → 分红不可持续 ⚠️
```

**茅台**：
- 自由现金流395亿
- 分红300亿
- 分红可持续 ✅

---

### 3. 抵抗风险能力

```
自由现金流大 = 抗风险能力强 ✅
```

| 公司 | 自由现金流 | 抗风险能力 |
|------|-----------|-----------|
| **茅台** | 395亿 | 极强 ✅ |
| **普通公司** | 10亿 | 一般 |
| **亏损公司** | -50亿 | 很弱 ❌ |

---

## 自由现金流与公司类型

### 现金奶牛型

```
经营现金流大，资本开支小 → 自由现金流大 ✅
```

**例子**：茅台

| 项目 | 金额（亿） |
|------|-----------|
| 经营现金流 | 445 |
| 资本开支 | 50 |
| **自由现金流** | **395** |

**特征**：
- 自由现金流极大
- 可以大额分红
- 抵抗风险能力强

---

### 成长型

```
经营现金流中等，资本开支大 → 自由现金流小或负
```

**例子**：某科技公司

| 项目 | 金额（亿） |
|------|-----------|
| 经营现金流 | 50 |
| 资本开支 | 80 |
| **自由现金流** | **-30** |

**特征**：
- 快速扩张期
- 需要融资
- 未来可能产生大量自由现金流

---

### 烧钱型

```
经营现金流为负，资本开支大 → 自由现金流大额为负 ❌
```

**例子**：某创业公司

| 项目 | 金额（亿） |
|------|-----------|
| 经营现金流 | -20 |
| 资本开支 | -30 |
| **自由现金流** | **-50** |

**特征**：
- 持续烧钱
- 需要大量融资
- 生存风险高

---

## 自由现金流收益率

### 定义

```
自由现金流收益率 = 自由现金流 ÷ 市值 × 100%
```

### 收益率标准

| 收益率 | 评价 |
|--------|------|
| > 10% | 很好 ✅ |
| 5%-10% | 良好 |
| 3%-5% | 一般 |
| < 3% | 较低 |

**茅台例子**（估算）：

| 项目 | 数值 |
|------|------|
| 自由现金流 | 395亿 |
| 市值 | 2.3万亿 |
| **自由现金流收益率** | **约1.7%** |

**为什么这么低？**
- 股价太高（市梦率）
- 虽然自由现金流大
- 但相对于市值，收益率不高

---

## 巴菲特与自由现金流

### 巴菲特最看重的指标

> "一家公司的价值，等于其未来自由现金流的折现值。"

**巴菲特的逻辑**：
1. 净利润可以操纵
2. 自由现金流很难操纵
3. 买股票就是买自由现金流

**巴菲特喜欢的公司**：
- 自由现金流持续为正 ✅
- 自由现金流稳定增长 ✅
- 不需要大量资本开支 ✅

**茅台**：完全符合巴菲特的标准！

---

## 自由现金流的"陷阱"

### 陷阱1：资本开支不足

**现象**：为了提高自由现金流，减少必须的资本开支

**后果**：
- 短期自由现金流很好看
- 长期竞争力下降
- 设备老化、技术落后

**如何识别**：
- 看资本开支是否远低于折旧
- 对比同行的资本开支率

---

### 陷阱2：自由现金流波动大

**原因**：
- 资本开支不稳定
- 经营现金流波动大

**例子**：周期性行业（钢铁、航运）

| 年份 | 经营现金流 | 资本开支 | 自由现金流 |
|------|-----------|----------|-----------|
| 第1年 | 100 | 30 | 70 |
| 第2年 | 20 | 30 | -10 |
| 第3年 | 150 | 30 | 120 |

**分析**：
- 自由现金流波动极大
- 行业周期性强
- 难以预测

---

## 如何用自由现金流分析公司

### 第一步：看自由现金流是否为正

```
持续为正 ✅ = 真正赚钱
持续为负 ❌ = 烧钱（创业期除外）
```

---

### 第二步：看自由现金流趋势

```
持续增长 ✅ = 造血能力增强
持续下降 ⚠️ = 造血能力恶化
波动大 = 周期性行业
```

---

### 第三步：看自由现金流 vs 净利润

| 情况 | 评价 |
|------|------|
| 自由现金流 ≈ 净利润 | 正常 ✅ |
| 自由现金流 > 净利润 | 很好 ✅ |
| 自由现金流 < 净利润 | 一般 ⚠️ |
| 自由现金流 << 净利润 | 差 ❌ |

**茅台**：自由现金流 ≈ 净利润（395 vs 400）

---

### 第四步：看自由现金流用途

| 用途 | 评价 |
|------|------|
| 分红 | 回报股东 ✅ |
| 回购 | 提高EPS ✅ |
| 还债 | 降低杠杆 ✅ |
| 收购 | 要看质量 ⚠️ |
| 存银行 | 有点浪费 💤 |

**茅台**：主要是分红和存银行

<summary>点击查看答案</summary>

自由现金流 = 100 - 80 = 20亿

分析：自由现金流20亿，说明公司"真正"赚了20亿现金，可以用来分红、回购或还债。

</details>

---

**问题2**：茅台为什么自由现金流这么高？

<details>
<summary>点击查看答案</summary>

- 资本开支相对小（50亿）

- 现金奶牛的典型特征
- 不需要大量投入就能产生大量现金

</details>

**问题3**：B公司净利润100亿，自由现金流20亿。这说明了什么？

<details>
<summary>点击查看答案</summary>

利润质量一般 ⚠️
- 净利润100亿
- 但自由现金流只有20亿
- 说明80亿利润是"纸面富贵"
- 可能是大量应收账款或存货占用现金
- 自由现金流才是"真金白银"

</details>
