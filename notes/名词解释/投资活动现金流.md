# 投资活动现金流（Investing Cash Flow）

## 定义

**投资活动现金流** = 公司**长期投资**相关的现金收支。

简单来说：**公司买设备、建厂房、做长期投资花的钱和收回的钱 —— 决定公司未来增长能力的投资。**

---

## 核心公式

```
投资活动净现金流 = 投资活动现金流入 - 投资活动现金流出
```

---

## 投资活动包括什么？

### 现金流入（钱进来了）

| 项目 | 说明 | 例子 |
|------|------|------|
| **收回投资收到的现金** | 卖股票、债券收回的钱 | 卖理财收回本金 |
| **取得投资收益收到的现金** | 分红、利息 | 收到股票分红 |
| **处置资产收到的现金** | 卖厂房、设备 | 卖旧设备 |
| **处置子公司收到的现金** | 卖子公司 | 卖掉不赚钱的子公司 |

**茅台**：主要流入是"取得投资收益"（理财收益）

---

### 现金流出（钱花出去了）

| 项目 | 说明 | 例子 |
|------|------|------|
| **购建固定资产支付现金** | 买设备、建厂房 | 建新酒厂 |
| **投资支付现金** | 买股票、债券、理财 | 买理财 |
| **并购子公司支付现金** | 买公司 | 并购同行业公司 |

**茅台**：主要是"购建固定资产"（建酒厂）和"投资支付现金"（买理财）

---

## 投资活动现金流的特点

### 通常是负数

```
好公司 = 投资现金流通常是负数 ✅
```

**为什么？**

| 原因 | 说明 |
|------|------|
| **成长需要投入** | 建厂房、买设备 |
| **扩张需要钱** | 开新店、并购 |
| **负数正常** | 说明公司在投资未来 |

**茅台**：每年投资现金流都是负数（在建新酒厂）

---

### 什么时候是正数？

| 情况 | 说明 |
|------|------|
| **成熟期公司** | 不需要大额投资 |
| **收缩期公司** | 在卖资产 |
| **衰退公司** | 变卖资产续命 |

**例子**：某钢铁厂

| 项目 | 金额（亿） |
|------|-----------|
| 卖厂房收入 | +50 |
| 买设备支出 | -5 |
| 投资现金流 | +45 |

**这不一定好事**：可能在收缩业务

---

## 投资活动与公司生命周期

### 创业期

| 经营现金流 | 投资现金流 | 筹资现金流 | 状态 |
|-----------|-----------|-----------|------|
| 负 | 负（大量投入） | 正（融资） | 烧钱扩张 |

**例子**：早期的京东

---

### 成长期

| 经营现金流 | 投资现金流 | 筹资现金流 | 状态 |
|-----------|-----------|-----------|------|
| 正 | 负（持续投入） | 正或负 | 快速扩张 |

**例子**：现在的拼多多（部分）

---

### 成熟期

| 经营现金流 | 投资现金流 | 筹资现金流 | 状态 |
|-----------|-----------|-----------|------|
| 正（大） | 负（适中） | 负（分红） | 现金奶牛 |

**例子**：茅台

---

### 衰退期

| 经营现金流 | 投资现金流 | 筹资现金流 | 状态 |
|-----------|-----------|-----------|------|
| 负或正 | 正（卖资产） | 负（还债） | 收缩 |

**例子**：某些传统零售

---

## 茅台的投资活动现金流

### 茅台投资现金流（2023年估算）

| 项目 | 金额（亿元） | 说明 |
|------|-------------|------|
| **流入** | | |
| 收到投资收益 | +5 | 理财收益 |
| 处置资产收入 | +1 | 卖旧设备 |
| **流入合计** | **+6** | |
| | | |
| **流出** | | |
| 购建固定资产 | -50 | 建新酒厂 |
| 投资支付现金 | -100 | 买理财 |
| **流出合计** | **-150** | |
| | | |
| **投资现金流净额** | **-144** | 扩张期，正常 |

**分析**：
- 茅台在扩张（建酒厂）
- 现金太多，买理财
- 净流出144亿
- 这是好公司正常现象

---

### 茅台为什么买这么多理财？

**原因**：

| 原因 | 说明 |
|------|------|
| **现金太多** | 每年几百亿现金流入 |
| **没地方花** | 主业不需要那么多投入 |
| **理财收益** | 比存银行好 |
| **灵活性** | 随时可以取出来 |

**茅台特点**：
- 现金流太好
- 理财只是"暂存"
- 未来可能分红或回购

---

## 投资活动现金流的"危险信号"

### 信号1：投资持续为正

```
投资现金流持续为正 → 可能是收缩期 ❌
```

**例外**：金融公司（投资是主业）

---

### 信号2：投资支出远超经营现金流

```
投资支出 >> 经营现金流 → 激进扩张 ⚠️
```

**例子**：某创业公司

| 项目 | 金额（亿） |
|------|-----------|
| 经营现金流 | +5 |
| 投资现金流 | -50 |

**问题**：
- 扩张过于激进
- 需要大量融资
- 资金链断裂风险

---

### 信号3：投资效率低

```
投资增加，但收入不增长 → 投资效率低 ⚠️
```

**例子**：某制造公司

| 年份 | 累计投资（亿） | 收入（亿） |
|------|--------------|-----------|
| 第1年 | 10 | 100 |
| 第2年 | 20 | 105 |
| 第3年 | 30 | 108 |

**问题**：
- 投了大量钱
- 收入增长缓慢
- 投资回报率低

---

## 资本开支（Capital Expenditure）

### 定义

```
资本开支 = 购建固定资产支付的现金
```

**茅台**：每年约50亿（建酒厂、买设备）

---

### 资本开支率

```
资本开支率 = 资本开支 ÷ 经营现金流
```

| 比率 | 评价 |
|------|------|
| < 30% | 很低，现金奶牛 ✅（茅台约10%） |
| 30%-50% | 适中 |
| 50%-100% | 较高 |
| > 100% | 很高，需要融资 ⚠️ |

**茅台**：50 ÷ 445 ≈ 11%（极低）

---

### 维持性资本开支 vs 扩张性资本开支

| 类型 | 说明 |
|------|------|
| **维持性** | 维持现有产能（更新设备） |
| **扩张性** | 扩大产能（建新厂） |

**分析要点**：
- 维持性资本开支是必须的
- 扩张性资本开支是可选的
- 看资本开支的必要性

---

## 如何用投资现金流分析公司

### 第一步：看投资现金流是否合理

```
负数但合理 ✅ = 正常投资
负数过大 ⚠️ = 激进扩张
正数 ❌ = 可能是收缩期
```

---

### 第二步：看资本开支率

```
资本开支率 = 资本开支 ÷ 经营现金流

< 30% ✅ = 现金奶牛
30%-50% = 适中
> 100% ⚠️ = 激进扩张
```

---

### 第三步：看投资效率

```
投资增长 → 收入应该增长
```

| 情况 | 评价 |
|------|------|
| 投资增加，收入增加 | 投资有效 ✅ |
| 投资增加，收入不变 | 投资低效 ⚠️ |
| 投资增加，收入下降 | 投资失败 ❌ |

---

### 第四步：看投资方向

| 投资方向 | 评价 |
|----------|------|
| 主业相关 | 好 ✅（茅台建酒厂） |
| 多元化 | 要谨慎 ⚠️ |
| 理财 | 暂时性，中性 |

## CFA 考试重点

### 考试频率：⭐⭐⭐⭐

### 必须掌握：

1. **投资现金流的定义**
   - 长期投资相关的现金收支

2. **投资现金流通常为负**
   - 成长公司需要投资
   - 负数是正常的

3. **资本开支**
   - 资本开支 = 购建固定资产
   - 资本开支率 = 资本开支 ÷ 经营现金流

4. **投资效率**
   - 投资增长应该带来收入增长

## 相关概念

| 概念 | 关系 |
|------|------|
| 经营活动现金流 | 经营+投资+筹资=总现金流 |
| 筹资活动现金流 | 经营+投资+筹资=总现金流 |
| 资本开支 | 投资现金流的主要流出 |
| [自由现金流](./自由现金流.md) | 经营现金流 - 资本开支 |
