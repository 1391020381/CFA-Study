# 资产 (Assets)

## 定义

**资产** = 公司拥有的、能够**带来未来经济利益**的资源。

简单来说：**公司能用的"钱"和"东西" —— 无论是现在就能用的，还是未来能产生价值的。**

---

## 核心特征

一个要被确认为资产的项目，必须同时满足：

| 特征 | 说明 |
|------|------|
| **1. 过去的交易形成** | 是公司通过买卖、生产等实际交易获得的 |
| **2. 被公司控制** | 公司有权支配这个资源 |
| **3. 能带来未来经济利益** | 未来能直接或间接产生现金流入 |

**例子**：
- ✅ 茅台的库存酒 → 能卖钱 → 是资产
- ✅ 茅台的厂房 → 能生产酒 → 是资产
- ✅ 茅台的商标 → 能带来品牌溢价 → 是资产
- ❌ 空气 → 虽然重要但无法控制 → 不是资产
- ❌ 过期变质的原料 → 无法带来利益 → 不是资产

---

## 资产的分类

### 按流动性分类（最重要）

```
资产
├─ 流动资产（一年内能变现）
│  ├─ 货币资金
│  ├─ 应收账款
│  ├─ 存货
│  └─ 交易性金融资产
│
└─ 非流动资产（长期使用）
   ├─ 固定资产
   ├─ 无形资产
   ├─ 长期股权投资
   └─ 长期待摊费用
```

---

### 流动资产（Current Assets）

**定义**：预计在**一年内或一个营业周期内**能变现、出售或耗用的资产。

| 科目 | 含义 | 例子 |
|------|------|------|
| **货币资金** | 现金、银行存款 | 茅台账上的690亿现金 |
| **应收账款** | 赊销后客户欠的钱 | 经销商欠茅台的酒钱 |
| **存货** | 准备卖的商品、原料 | 茅台的库存酒、粮食 |
| **交易性金融资产** | 短期持有的股票、基金 | 公司用闲钱买的理财 |
| **预付账款** | 预先付给供应商的钱 | 预付包装厂定金 |

**流动资产的核心**：能快速变成现金，应对短期支付需求。

---

### 非流动资产（Non-current Assets）

**定义**：预计持有**超过一年**的资产，主要用于长期生产经营。

| 科目 | 含义 | 例子 |
|------|------|------|
| **固定资产** | 厂房、设备、车辆 | 茅台的酿酒车间 |
| **无形资产** | 专利、商标、土地使用权 | 茅台的"茅台"商标 |
| **长期股权投资** | 长期持有的其他公司股份 | 茅台持有某子公司80%股份 |
| **商誉** | 并购时支付的溢价 | 买公司时多付的钱 |
| **长期待摊费用** | 需要长期分摊的费用 | 装修费分摊5年 |

**非流动资产的核心**：创造公司长期价值，但不容易变现。

---

## 常见资产科目详解

### 1. 货币资金

```
货币资金 = 库存现金 + 银行存款 + 其他货币资金
```

| 现金水平 | 含义 |
|----------|------|
| 很多（>20%） | 现金牛，但可能资金利用不足 |
| 适中（10-20%） | 健康水平 |
| 很少（<5%） | 资金紧张，可能要借钱 |

**茅台特点**：现金多（690亿），因为预收账款多，经销商先打款。

---

### 2. 应收账款

```
应收账款 = 赊销产生的债权
```

| 应收账款情况 | 含义 |
|--------------|------|
| 很少或为0 | 产品抢手，先款后货（茅台）✅ |
| 适中 | 正常商业信用 |
| 过多（>收入30%） | 产品难卖，大量赊销 ⚠️ |

**风险**：
- 客户可能不还钱（坏账）
- 占用公司资金
- 利润质量差

---

### 3. 存货

```
存货 = 原材料 + 在产品 + 库存商品 + 周转材料
```

| 存货情况 | 含义 |
|----------|------|
| 茅台的存货 | 越陈越值钱 ✅ |
| 普通商品存货 | 放久了会过期/贬值 ⚠️ |
| 电子产品存货 | 技术更新快，贬值快 ⚠️ |

**茅台特点**：
- 存货418亿（2023年）
- 主要是基酒（存5年才卖）
- 存货是资产，不是负担

---

### 4. 固定资产

```
固定资产 = 原值 - 累计折旧 = 账面价值
```

| 行业 | 固定资产特点 |
|------|--------------|
| 钢铁、电力 | 固定资产占比高（重资产） |
| 互联网、服务 | 固定资产占比低（轻资产） |
| 茅台 | 适中（厂房+设备） |

**重资产 vs 轻资产**：
- **重资产**：赚钱难（折旧高），但壁垒高
- **轻资产**：扩张快，但竞争激烈

---

### 5. 无形资产

```
无形资产 = 专利 + 商标 + 土地使用权 + 软件等
```

| 类型 | 例子 |
|------|------|
| 可辨认无形资产 | 专利、商标、软件 |
| 不可辨认无形资产 | 商誉（只在并购时产生） |

**茅台的无形资产**：
- "茅台"商标（价值万亿，但账上可能只记录注册费）
- 土地使用权
- 酿酒工艺（不体现在报表上）

---

## 资产质量分析

### 好资产 vs 坏资产

| 类型 | 好资产 ✅ | 坏资产 ❌ |
|------|-----------|-----------|
| 现金类 | 能直接用 | 被冻结、受限 |
| 应收账款 | 能收回 | 变成坏账 |
| 存货 | 能卖出去 | 积压、过期 |
| 固定资产 | 在用、高效 | 闲置、落后 |
| 无形资产 | 真有价值 | 估值虚高 |

---

## 资产结构分析

### 典型公司资产结构

| 公司类型 | 流动资产 | 非流动资产 | 特点 |
|----------|----------|------------|------|
| **茅台** | 现金多 | 存货适中 | 现金牛 |
| **零售商** | 存货多 | 固定资产多 | 周转快 |
| **互联网** | 现金多 | 无形资产多 | 轻资产 |
| **钢铁厂** | 存货适中 | 固定资产极高 | 重资产 |

## 资产负债表中的位置

```
资产负债表
│
├─ 资产（钱的用途）
│  ├─ 流动资产
│  │  ├─ 货币资金
│  │  ├─ 应收账款
│  │  └─ 存货
│  │
│  └─ 非流动资产
│     ├─ 固定资产
│     └─ 无形资产
│
├─ 负债（钱从哪来 - 借的）
│
└─ 所有者权益（钱从哪来 - 股东的）
```

**记住**：资产 = 负债 + 所有者权益

## CFA 考试重点

### 考试频率：⭐⭐⭐⭐⭐

### 必须掌握：

1. **资产的定义**：能带来未来经济利益

2. **流动 vs 非流动**
   - 流动资产：一年内变现
   - 非流动资产：长期持有

3. **关键科目**
   - 货币资金、应收账款、存货
   - 固定资产、无形资产

4. **资产质量判断**
   - 应收账款周转率
   - 存货周转率
   - 固定资产周转率
