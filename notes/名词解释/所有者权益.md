# 所有者权益 (Owner's Equity)

## 定义

**所有者权益** = 公司资产**扣除负债后**，**股东真正拥有**的净资产。

简单来说：**公司属于股东的那部分价值 —— 股东投的钱 + 公司赚了没分掉的钱。**

---

## 核心公式

```
所有者权益 = 资产 - 负债
```

换个角度：
```
资产（钱的用途） = 负债（借的） + 所有者权益（股东的）
```

**理解**：
- 公司的钱有两个来源
  - 从别人那借的（负债）
  - 股东投的 + 公司赚的（权益）

---

## 所有者权益的构成

```
所有者权益
├─ 股本（实收资本）
│  └─ 股东投入的本金
│
├─ 资本公积
│  └─ 股东超额投入的钱
│
├─ 留存收益
│  ├─ 盈余公积（强制留存）
│  └─ 未分配利润（可分配利润）
│
├─ 其他综合收益
│  └─ 某些资产的价值变动
│
└─ 少数股东权益
   └─ 子公司中不属于母公司的部分
```

---

## 主要科目详解

### 1. 股本（实收资本）

```
股本 = 股东投入的本金（按面值计算）
```

| 项目 | 说明 |
|------|------|
| **面值** | 股票票面金额（中国通常是1元） |
| **股数** | 发行的股票数量 |
| **股本** = 面值 × 股数 | |

**例子**：
- 茅台发行12.6亿股，每股面值1元
- 股本 = 12.6亿元

**为什么面值这么小？**
- 面值只是法律概念
- 真正的价值是股价（比如茅台股价1800元）

---

### 2. 资本公积

```
资本公积 = 股东投入超过面值的部分
```

**来源**：
1. **溢价发行**：发行价 > 面值
2. **资产增值**：资产评估增值
3. **其他**：捐赠等

**例子**：
- 茅台发行新股，发行价50元，面值1元
- 其中：1元进股本，49元进资本公积

**资本公积的用途**：
- 转增股本（送股）
- 不能直接分红（但转增后可以分红）

---

### 3. 留存收益

```
留存收益 = 公司历年赚了但没分给股东的钱
```

**构成**：
| 项目 | 说明 |
|------|------|
| **盈余公积** | 按法律规定必须留存的（通常10%） |
| **未分配利润** | 可以继续投资或分红的钱 |

**茅台例子**：
- 2023年净利润700亿
- 分红300亿
- 留存收益增加400亿

**留存收益的意义**：
- 公司扩大生产的资金来源
- 不用付利息的"资金"
- 说明公司盈利能力强

---

### 4. 其他综合收益

```
其他综合收益 = 某些资产价值变动（还没实现）
```

**例子**：
- 持有的股票升值了（但还没卖）
- 汇率变动导致外币资产价值变化
- 可供出售金融资产的价值变动

**特点**：
- 计入权益，但不算净利润
- 实现后才转入利润

---

### 5. 少数股东权益

```
少数股东权益 = 子公司中不属于母公司的权益
```

**例子**：
- 茅台持有某子公司80%股份
- 该子公司净资产100亿
- 少数股东权益 = 20亿

---

## 所有者权益与股息

### 股息来源

```
股息只能从"未分配利润"中分配！
```

| 项目 | 能分红吗？ |
|------|-----------|
| 股本 | ❌ 不能（这是股东的投入） |
| 资本公积 | ❌ 不能（但有例外） |
| 盈余公积 | ❌ 一般不能 |
| 未分配利润 | ✅ 可以 |
| 本年净利润 | ✅ 可以 |

**分红限制**：
- 分红后，留存收益不能低于股本的25%
- 公司要有足够现金

---

## 所有者权益的意义

### 1. 股东的真实财富

```
每股净资产 = 所有者权益 ÷ 总股本
```

| 项目 | 数值（茅台2023年） |
|------|-------------------|
| 所有者权益 | 2090亿元 |
| 总股本 | 12.6亿股 |
| 每股净资产 | 166元 |

**vs 股价**：
- 茅台股价约1800元
- 市净率（PB）= 1800 ÷ 166 ≈ 11倍

---

### 2. 举债能力

```
所有者权益越多，举债能力越强
```

| 所有者权益 | 意味着 |
|-----------|--------|
| 很多 | 银行愿意借钱（风险小） |
| 很少 | 银行不愿意借钱（风险大） |

**银行视角**：
- 所有者权益 = "安全垫"
- 如果亏损，先亏权益，后亏债权

---

### 3. ROE（净资产收益率）

```
ROE = 净利润 ÷ 所有者权益
```

| ROE水平 | 评价 |
|---------|------|
| > 20% | 优秀 ✅（茅台约25%） |
| 10%-20% | 良好 |
| < 10% | 一般 |

**巴菲特最看重的指标！**

## 资产负债表中的位置

```
资产负债表
│
├─ 资产（钱的用途）
│
├─ 负债（钱从哪来 - 借的）
│
└─ 所有者权益（钱从哪来 - 股东的）
   ├─ 股本
   ├─ 资本公积
   ├─ 留存收益
   ├─ 其他综合收益
   └─ 少数股东权益
```

**记住**：资产 = 负债 + 所有者权益

---

## 所有者权益变动表

这是**第四张财务报表**！

| 项目 | 说明 |
|------|------|
| 期初余额 | 年初的所有者权益 |
| + 本年净利润 | 增加权益 |
| - 股息分配 | 减少权益 |
| + 股东增资 | 增加权益 |
| = 期末余额 | 年末的所有者权益 |

---

## 典型公司权益结构

| 公司类型 | 股本占比 | 留存收益占比 | 特点 |
|----------|----------|--------------|------|
| **茅台** | 很小（<1%） | 很大（>95%） | 盈利能力强，分红多 |
| **创业公司** | 较大 | 小或负 | 还在投入期 |
| **银行** | 适中 | 适中 | 监管要求 |
| **成熟企业** | 小 | 大 | 现金牛 |

## 账面价值 vs 市场价值

### 账面价值（Book Value）

```
账面价值 = 所有者权益
```

- 来自资产负债表
- 基于历史成本
- 不包括"隐形资产"

### 市场价值（Market Value）

```
市场价值 = 股价 × 总股本
```

- 来自股票市场
- 基于未来预期
- 包括所有价值

**茅台对比**（2023年）：
| 项目 | 数值 |
|------|------|
| 账面价值 | 2090亿元 |
| 市场价值 | 约2.3万亿元 |
| 市净率（PB） | 约11倍 |

**为什么差这么多？**
- 品牌价值（万亿级）
- 稀缺性（独家生产）
- 盈利能力（毛利率90%+）

## CFA 考试重点

### 考试频率：⭐⭐⭐⭐⭐

### 必须掌握：

1. **所有者权益的定义**：资产 - 负债

2. **权益构成**
   - 股本、资本公积、留存收益
   - 各部分的来源

3. **关键指标**
   - 每股净资产 = 权益 ÷ 股本
   - ROE = 净利润 ÷ 权益
   - 市净率（PB）= 股价 ÷ 每股净资产

4. **留存收益**
   - 来源：净利润 - 分红
   - 用途：再投资、分红
