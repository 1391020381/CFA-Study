# 负债 (Liabilities)

## 定义

**负债** = 公司**欠别人的**、需要在**未来偿还**的债务。

简单来说：**公司借来用的"钱" —— 无论是欠供应商的、欠银行的，还是欠员工的。**

---

## 核心特征

一个要被确认负债的项目，必须同时满足：

| 特征 | 说明 |
|------|------|
| **1. 过去的交易形成** | 是因为过去的业务产生的义务 |
| **2. 现时的义务** | 现在就必须承担责任 |
| **3. 需要未来偿付** | 未来要流出经济利益（现金、商品、劳务） |

**例子**：
- ✅ 向银行贷款 → 欠银行钱 → 是负债
- ✅ 赊购原料 → 欠供应商钱 → 是负债
- ✅ 欠员工工资 → 欠员工钱 → 是负债
- ✅ 收客户预付款 → 欠客户货 → 是负债
- ❌ 计划未来购买 → 还没发生 → 不是负债

---

## 负债的分类

### 按到期时间分类（最重要）

```
负债
├─ 流动负债（一年内要还）
│  ├─ 应付账款
│  ├─ 短期借款
│  ├─ 应付职工薪酬
│  └─ 应交税费
│
└─ 非流动负债（一年后还）
   ├─ 长期借款
   ├─ 应付债券
   └─ 长期应付款
```

---

### 流动负债（Current Liabilities）

**定义**：预计在**一年内或一个营业周期内**需要偿还的债务。

| 科目 | 含义 | 例子 |
|------|------|------|
| **应付账款** | 欠供应商的钱 | 茅台欠包装厂的钱 |
| **短期借款** | 一年内到期的贷款 | 银行流动资金贷款 |
| **应付票据** | 开出的商业承兑汇票 | 给供应商的期票 |
| **预收账款** | 预先收客户的款 | 经销商预付茅台的酒钱 |
| **应付职工薪酬** | 欠员工的工资 | 月底应付的工资 |
| **应交税费** | 欠国家的税 | 应交但未交的增值税 |

**流动负债的核心**：短期内要还，考验公司短期偿债能力。

---

### 非流动负债（Non-current Liabilities）

**定义**：偿还期限在**一年以上**的长期债务。

| 科目 | 含义 | 例子 |
|------|------|------|
| **长期借款** | 一年后到期的贷款 | 银行项目贷款 |
| **应付债券** | 公司发行的债券 | 茅台发行的5年期债券 |
| **长期应付款** | 长期的应付款 | 融资租赁应付款 |
| **递延所得税负债** | 以后要交的税 | 会计利润>税法利润 |

**非流动负债的核心**：长期资金来源，用于长期投资。

---

## 常见负债科目详解

### 1. 应付账款

```
应付账款 = 赊购产生的债务
```

| 应付账款情况 | 含义 |
|--------------|------|
| 适中 | 正常商业信用 |
| 过多（>资产30%） | 资金紧张，大量拖欠 ⚠️ |
| 很少 | 现金充足或议价能力弱 |

**茅台特点**：
- 应付账款120亿
- 可以占用供应商资金（话语权强）

---

### 2. 预收账款（合同负债）

```
预收账款 = 提前收客户的钱，以后要交货
```

| 预收账款情况 | 含义 |
|--------------|------|
| 很多 | 产品抢手！客户先打款排队 ✅ |
| 很少或为0 | 产品一般，先货后款 |

**茅台特点**：
- 预收账款150亿（2023年）
- 经销商先打款，等半年后拿货
- 这是"无息贷款"！

**关键理解**：
- 预收账款是负债（欠客户货）
- 但这是**好负债**（免费资金）

---

### 3. 短期借款 vs 长期借款

| 类型 | 期限 | 用途 | 风险 |
|------|------|------|------|
| **短期借款** | < 1年 | 日常周转 | 高（一年内要还） |
| **长期借款** | > 1年 | 扩建项目 | 低（慢慢还） |

**风险提示**：
- ⚠️ 短期借款买长期资产 = **期限错配**
- 例如：借1年期贷款建10年才能收回的工厂
- 危险：贷款到期时，项目还没回款

---

### 4. 应付债券

```
应付债券 = 公司发行的债券
```

| 特点 | 说明 |
|------|------|
| 利息固定 | 每年付息 |
| 到期还本 | 到期日一次性还本金 |
| 可以交易 | 债券可以在市场上买卖 |

**为什么发债券而不是找银行贷款？**
- 债券利率可能更低
- 期限更长（10年、20年）
- 不用抵押品（信用好的公司）

---

## 负债质量分析

### 好负债 vs 坏负债

| 类型 | 好负债 ✅ | 坏负债 ❌ |
|------|-----------|-----------|
| 经营性负债 | 预收账款、应付账款 | 过度拖欠 |
| 金融性负债 | 长期低息贷款 | 高息短期贷款 |
| 有息负债 | 能产生更高收益 | 资金空转 |

---

### 负债结构健康度

| 指标 | 健康 ⚕️ | 不健康 🚨 |
|------|----------|-----------|
| 资产负债率 | < 50% | > 70% |
| 流动比率 | > 2 | < 1 |
| 有息负债/总负债 | < 30% | > 60% |
| 短期借款占比 | 低 | 高 |

---

## 负债的"好"与"坏"

### 好负债（无息负债）

```
好负债 = 免费使用别人的钱
```

| 类型 | 说明 |
|------|------|
| **预收账款** | 客户先给钱，后给货（茅台） |
| **应付账款** | 占用供应商资金 |
| **应付职工薪酬** | 先干活，后发工资 |

**为什么是好负债？**
- 不用付利息
- 期限灵活
- 说明公司话语权强

---

### 坏负债（有息负债）

```
坏负债 = 要付利息的债务
```

| 类型 | 说明 |
|------|------|
| **银行贷款** | 要付利息 |
| **应付债券** | 要付利息 |
| **其他借款** | 要付利息 |

**不一定是"坏"的**：
- 如果借钱的收益率 > 利息率 → 加杠杆赚钱
- 如果借钱的收益率 < 利息率 → 恶性循环

---

## 财务杠杆：负债的两面性

### 杠杆放大收益

```
例子：A公司投资回报率15%，借款利率5%

情况1：不用杠杆
- 投入100万 → 赚15万 → 收益率15%

情况2：用杠杆（借50万）
- 自有50万 + 借50万 = 投入100万
- 赚15万 - 利息2.5万 = 净赚12.5万
- 收益率 = 12.5万 ÷ 50万 = 25% ✅ 放大了！
```

---

### 杠杆放大风险

```
例子：A公司投资回报率3%，借款利率5%

情况1：不用杠杆
- 投入100万 → 赚3万 → 收益率3%

情况2：用杠杆（借50万）
- 赚3万 - 利息2.5万 = 净赚0.5万
- 收益率 = 0.5万 ÷ 50万 = 1% ❌ 收益率下降！

如果投资回报率 < 0（亏损）：
- 亏损 - 利息 = 双重打击！
```

## 资产负债表中的位置

```
资产负债表
│
├─ 资产（钱的用途）
│
├─ 负债（钱从哪来 - 借的）
│  ├─ 流动负债
│  │  ├─ 应付账款
│  │  ├─ 短期借款
│  │  └─ 预收账款
│  │
│  └─ 非流动负债
│     ├─ 长期借款
│     └─ 应付债券
│
└─ 所有者权益（钱从哪来 - 股东的）
```

**记住**：资产 = 负债 + 所有者权益

## 典型公司负债结构

| 公司类型 | 有息负债 | 无息负债 | 特点 |
|----------|----------|----------|------|
| **茅台** | 很少 | 很多（预收账款） | 现金牛，话语权强 ✅ |
| **房地产** | 很多 | 很多（预收房款） | 高杠杆经营 |
| **银行** | 很多（存款） | 很少 | 资金是负债 |
| **制造企业** | 适中 | 适中 | 正常水平 |

## CFA 考试重点

### 考试频率：⭐⭐⭐⭐⭐

### 必须掌握：

1. **负债的定义**：现时义务，需要未来偿付

2. **流动 vs 非流动**
   - 流动负债：一年内偿还
   - 非流动负债：长期债务

3. **关键科目**
   - 应付账款、预收账款
   - 短期借款、长期借款

4. **负债比率分析**
   - 资产负债率 = 总负债 ÷ 总资产
   - 流动比率 = 流动资产 ÷ 流动负债
   - 速动比率 = (流动资产 - 存货) ÷ 流动负债
