# 营业成本（Cost of Goods Sold, COGS）

## 定义

**营业成本**（也叫主营业务成本）= 公司销售商品或提供劳务所发生的**直接成本**。

简单来说：**生产产品/提供服务直接花掉的钱 —— 每卖出一件商品，就产生的成本。**

---

## 核心公式

```
营业成本 = 直接材料 + 直接人工 + 制造费用
```

---

## 营业成本的构成

### 1. 直接材料

```
直接材料 = 直接用于产品生产的原材料
```

| 公司 | 直接材料 |
|------|----------|
| **茅台** | 粮食（高粱、小麦）、水、包装材料 |
| **汽车公司** | 钢材、玻璃、轮胎、芯片 |
| **服装公司** | 面料、纽扣、拉链 |

**特点**：每多生产一件，就多一份材料成本

---

### 2. 直接人工

```
直接人工 = 直接参与生产的人员工资
```

| 公司 | 直接人工 |
|------|----------|
| **茅台** | 酿酒师傅工资 |
| **汽车公司** | 生产线工人工资 |
| **软件公司** | 程序员工资 |

**特点**：生产越多，人工成本越高（但可能有规模效应）

---

### 3. 制造费用

```
制造费用 = 间接生产成本
```

| 包括 | 例子 |
|------|------|
| **折旧** | 机器设备、厂房折旧 |
| **水电** | 生产用电、用水 |
| **维修** | 设备维修费 |
| **车间管理** | 车间主任工资 |

**特点**：间接但必要，需要分摊到产品

---

## 营业成本 vs 期间费用

| 类型 | 内容 | 与销量关系 |
|------|------|-----------|
| **营业成本** | 直接材料+人工+制造费用 | 正相关 |
| **期间费用** | 销售、管理、研发、财务 | 相对固定 |

**例子**：茅台卖1瓶酒 vs 卖1000瓶酒

| 项目 | 卖1瓶 | 卖1000瓶 |
|------|-------|----------|
| 粮食成本 | 10元 | 1万元 |
| 厂房折旧 | 分摊100元 | 分摊10万元 |
| 销售费用 | 5元（基本固定） | 5000元（可能不线性增加） |
| 管理费用 | 10元（基本固定） | 1万元（基本固定） |

## 茅台的营业成本

### 茅台成本结构（估算）

| 成本项目 | 金额（估算） | 占成本比例 |
|----------|-------------|-----------|
| 直接材料 | 40亿 | 60% |
| 直接人工 | 10亿 | 15% |
| 制造费用（含折旧） | 15亿 | 25% |
| **合计** | **约65亿** | 100% |

**茅台特点**：
- 成本占收入比例很低（约10%）
- 毛利率高达90%+
- 粮食成本占比最大

## 成本核算方法

### 存货计价方法

当原材料价格波动时，如何计算成本？

| 方法 | 说明 | 通胀时影响 |
|------|------|-----------|
| **FIFO（先进先出）** | 先买的先卖出 | 成本偏低，利润偏高 |
| **LIFO（后进先出）** | 后买的先卖出 | 成本偏高，利润偏低（美国可用） |
| **加权平均** | 按平均价计算 | 介于两者之间 |

**中国会计准则**：多用先进先出或加权平均

---

### 成本分摊

```
单个产品成本 = 总成本 ÷ 产量
```

**茅台例子**：
- 年生产成本：65亿
- 年产量：5万吨
- 每吨成本：13万元

---

## 营业成本的分析

### 1. 成本收入比

```
成本收入比 = 营业成本 ÷ 营业收入
```

| 比例 | 评价 | 例子 |
|------|------|------|
| < 30% | 极好 ✅ | 茅台约10% |
| 30%-50% | 很好 | 苹果约40% |
| 50%-70% | 一般 | 制造业 |
| > 70% | 很差 | 零售、加工 |

---

### 2. 成本增长率 vs 收入增长率

| 情况 | 含义 |
|------|------|
| 收入增长 > 成本增长 | 规模效应 ✅ |
| 收入增长 < 成本增长 | 成本失控 ⚠️ |

**茅台**：收入增长15%，成本增长约10% → 规模效应

---

### 3. 成本结构变化

| 变化 | 含义 |
|------|------|
| 原材料占比上升 | 原材料涨价或议价能力弱 |
| 人工占比上升 | 人工成本上涨或自动化程度低 |
| 制造费用占比上升 | 产能利用率低 |

---

## 成本操纵的"陷阱"

### 陷阱1：少结转成本

**手段**：把本该计入成本的费用"遗忘"

**例子**：
- 应该结转的存货不结转
- 虚增库存价值

**后果**：
- 成本低估
- 利润虚高
- 库存积压

---

### 陷阱2：费用资本化

**手段**：把应该计入成本/费用的支出变成"资产"

**例子**：
- 把正常研发费用资本化
- 把广告费资本化

**如何识别**：
- 看"开发支出"是否异常
- 对比同行研发费用率

## 成本优化策略

### 1. 规模效应

```
产量增加 → 单位固定成本下降
```

**例子**：
- 年产1000吨：每吨成本15万
- 年产10000吨：每吨成本11万
- 固定成本被分摊

---

### 2. 供应链优化

| 策略 | 例子 |
|------|------|
| 集中采购 | 降低原材料单价 |
| 长期合同 | 锁定价格 |
| 垂直整合 | 自己生产原材料 |

---

### 3. 技术升级

| 策略 | 效果 |
|------|------|
| 自动化 | 降低人工成本 |
| 工艺改进 | 降低材料消耗 |
| 设备更新 | 提高效率 |

## 学习检查

**问题1**：茅台卖1瓶酒的成本10元，卖1000瓶的成本是1万元吗？

<details>
<summary>点击查看答案</summary>

不一定！
- 直接材料（粮食）会线性增加
- 但制造费用（厂房折旧）是固定的
- 所以平均成本可能下降（规模效应）

</details>

---

**问题2**：为什么茅台的毛利率高达90%+？

<details>
<summary>点击查看答案</summary>

- 品牌极强，可以定高价
- 生产成本相对固定
- 原材料成本占比低
- 有强大的定价权

</details>

---

**问题3**：A公司收入增长20%，成本增长30%，这是什么信号？

<details>
<summary>点击查看答案</summary>

危险信号 ⚠️
- 成本增速超过收入增速
- 可能是原材料涨价
- 可能是管理效率下降
- 需要深入分析原因

</details>
