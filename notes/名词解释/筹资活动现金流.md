# 筹资活动现金流（Financing Cash Flow）

## 定义

**筹资活动现金流** = 公司**借钱或找股东要钱**相关的现金收支。

简单来说：**公司找钱（借钱、发股票）和还钱（还债、分红、回购）的现金流动 —— 公司的"输血"和"回馈"。**

---

## 核心公式

```
筹资活动净现金流 = 筹资活动现金流入 - 筹资活动现金流出
```

---

## 筹资活动包括什么？

### 现金流入（钱进来了）

| 项目 | 说明 | 例子 |
|------|------|------|
| **吸收投资收到现金** | 股东投钱 | IPO、增发股票 |
| **取得借款收到现金** | 找银行借钱 | 银行贷款 |
| **发行债券收到现金** | 发债券借的钱 | 公司债券 |

**创业公司**：主要流入是"吸收投资"（融资）
**茅台**：很少需要筹资（现金太多）

---

### 现金流出（钱花出去了）

| 项目 | 说明 | 例子 |
|------|------|------|
| **偿还债务支付现金** | 还银行钱 | 还贷款 |
| **分配股利支付现金** | 给股东分红 | 分红 |
| **回购股票支付现金** | 买回股票 | 回购注销 |

**茅台**：主要流出是"分配股利"（分红）

---

## 筹资活动与公司生命周期

### 创业期

| 经营现金流 | 投资现金流 | 筹资现金流 | 状态 |
|-----------|-----------|-----------|------|
| 负 | 负 | **正（融资）** | 烧钱扩张 |

**特点**：
- 需要大量资金
- 找VC、PE融资
- 筹资现金流为正

**例子**：早期的字节跳动

---

### 成长期

| 经营现金流 | 投资现金流 | 筹资现金流 | 状态 |
|-----------|-----------|-----------|------|
| 正 | 负 | **正或负** | 扩张 |

**特点**：
- 经营开始产生现金
- 但还不够投资
- 偶尔需要融资

**例子**：成长期的科技公司

---

### 成熟期（现金奶牛）

| 经营现金流 | 投资现金流 | 筹资现金流 | 状态 |
|-----------|-----------|-----------|------|
| 正（大） | 负（小） | **负（分红）** | 回报股东 |

**特点**：
- 经营现金很多
- 不需要借钱
- 给股东分红

**例子**：茅台

---

### 衰退期

| 经营现金流 | 投资现金流 | 筹资现金流 | 状态 |
|-----------|-----------|-----------|------|
| 负 | 正 | **负（还债）** | 收缩 |

**特点**：
- 主业不赚钱
- 靠卖资产还债
- 筹资现金流为负

**例子**：某些传统制造业

---

## 茅台的筹资活动现金流

### 茅台筹资现金流（2023年估算）

| 项目 | 金额（亿元） | 说明 |
|------|-------------|------|
| **流入** | | |
| 借款收到现金 | +10 | 少量借款 |
| **流入合计** | **+10** | |
| | | |
| **流出** | | |
| 偿还债务支付现金 | -10 | 还贷款 |
| 分配股利支付现金 | -300 | 给股东分红 |
| **流出合计** | **-310** | |
| | | |
| **筹资现金流净额** | **-300** | 慷慨分红 |

**分析**：
- 茅台主要流出是分红
- 不需要借钱（现金太多）
- 净流出300亿
- 这是现金奶牛的典型特征！

---

### 茅台为什么筹资现金流总是负数？

**原因**：

| 原因 | 说明 |
|------|------|
| **经营现金流很大** | 每年几百亿现金流入 |
| **不需要借钱** | 自有现金足够 |
| **慷慨分红** | 每年分红几百亿 |
| **偶尔回购** | 买回股票注销 |

**茅台特征**：
- 筹资现金流长期为负
- 说明现金极其充裕
- 是最好的"现金奶牛"！

---

## 分红 vs 回购

### 分红（Dividend）

```
分红 = 直接给股东发钱
```

| 优点 | 缺点 |
|------|------|
| 直接现金回报 | 要交税 |
| 可持续（看现金流） | 现金流出 |
| 股价不直接受影响 | |

**茅台**：每年分红约300亿+

---

### 回购（Share Repurchase）

```
回购 = 公司买回自己的股票并注销
```

| 优点 | 缺点 |
|------|------|
| 提高EPS | 需要现金 |
| 股价上涨 | 可能时机不好 |
| 税收优惠 | |

**苹果**：经常回购股票

---

### 茅台的策略

| 策略 | 说明 |
|------|------|
| **主要分红** | 每年分红300亿+ |
| **偶尔回购** | 较少（未来可能增加） |

**为什么茅台主要分红？**
- A股投资者喜欢分红
- 茅台现金流稳定
- 分红信号更明确

---

## 筹资活动现金流的"危险信号"

### 信号1：持续依赖融资

```
经营现金流为负，筹资现金流为正 → 持续"输血" ⚠️
```

**例子**：某创业公司

| 项目 | 金额（亿） |
|------|-----------|
| 经营现金流 | -20 |
| 投资现金流 | -30 |
| 筹资现金流 | +60 |

**问题**：
- 主业不赚钱
- 需要不断融资
- 一旦融资困难，资金链断裂

---

### 信号2：分红超过经营现金流

```
分红 > 经营现金流 → 不可持续 ⚠️
```

**例子**：某公司

| 项目 | 金额（亿） |
|------|-----------|
| 经营现金流 | 10 |
| 分红 | 15 |

**问题**：
- 分红超过产生现金的能力
- 需要借钱分红
- 不可持续

---

### 信号3：频繁增发股票

```
频繁增发 → 股权稀释 ⚠️
```

**问题**：
- 老股东权益被稀释
- EPS下降
- 可能是公司缺钱

---

## 股利支付率

### 定义

```
股利支付率 = 分红 ÷ 净利润 × 100%
```

### 支付率标准

| 支付率 | 评价 |
|--------|------|
| **0%** | 不分红（成长期或无能力） |
| **10%-30%** | 适中（留存部分发展） |
| **30%-60%** | 慷慨（茅台约50%） |
| **> 80%** | 很高（可能不可持续） |
| **> 100%** | 危险（借钱分红） ❌ |

**茅台**：约50%（很健康的水平）

---

## 如何用筹资现金流分析公司

### 第一步：看筹资现金流与经营现金流的关系

| 情况 | 评价 |
|------|------|
| 经营正，筹资负（分红） | 现金奶牛 ✅（茅台） |
| 经营负，筹资正（融资） | 烧钱扩张 ⚠️（创业公司） |
| 经营负，筹资负（还债） | 衰退收缩 ❌ |

---

### 第二步：看分红是否可持续

```
分红 ÷ 经营现金流 < 80% ✅ 可持续
分红 ÷ 经营现金流 > 100% ❌ 不可持续
```

---

### 第三步：看资本结构

| 情况 | 评价 |
|------|------|
| 股权融资为主 | 稳健，无债务压力 ✅ |
| 债权融资为主 | 激进，有债务压力 ⚠️ |
| 平衡 | 适中 |

---

### 第四步：看股权变化

| 情况 | 评价 |
|------|------|
| 回购注销 | 利好股东 ✅ |
| 增发新股 | 稀释股权 ⚠️ |
| 不变 | 正常 |

## 学习检查

**问题1**：A公司经营现金流100亿，筹资现金流-50亿（主要是分红）。这说明了什么？

<details>
<summary>点击查看答案</summary>

现金奶牛 ✅
- 主业赚钱（经营现金流100亿）
- 给股东分红（筹资现金流-50亿）
- 分红占经营现金流的50%，可持续
- 这是成熟好公司的特征

</details>

---

**问题2**：茅台为什么筹资现金流总是负数？

<details>
<summary>点击查看答案</summary>

- 茅台经营现金流很大（几百亿）
- 不需要借钱（自有现金足够）
- 每年分红几百亿
- 筹资现金流为负是好公司的特征
- 说明在回馈股东

</details>

---

**问题3**：B公司经营现金流-10亿，筹资现金流+20亿（主要是融资）。这是什么信号？

<details>
<summary>点击查看答案</summary>

创业/烧钱公司 ⚠️
- 主业不赚钱（经营现金流-10亿）
- 需要不断融资（筹资现金流+20亿）
- 靠"输血"维持
- 如果融资困难，可能资金链断裂

</details>
