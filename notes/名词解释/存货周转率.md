# 存货周转率（Inventory Turnover）

## 定义

**存货周转率** = 营业成本 ÷ 平均存货。

简单来说：**存货一年能周转几次（卖出去几次） —— 衡量存货管理效率和销售能力的核心指标。**

---

## 核心公式

```
存货周转率 = 营业成本 ÷ 平均存货

平均存货 = (期初存货 + 期末存货) ÷ 2
```

**简化计算**（常用）：
```
存货周转率 ≈ 营业成本 ÷ 期末存货
```

---

## 存货周转天数

```
存货周转天数 = 365 ÷ 存货周转率
```

**含义**：存货平均多少天能卖出去一次。

---

## 举例理解

### 茅台的存货周转（估算）

| 项目 | 金额（亿元） |
|------|-------------|
| 营业成本 | 70 |
| 期末存货 | 400 |
| **存货周转率** | **0.18次/年** |
| **存货周转天数** | **2024天**（约5.5年） |

**解读**：
- 茅台的存货一年周转0.18次
- 平均要5.5年才能卖出去
- 这看起来很糟糕？

**但这是茅台的特色！**
- 茅台的存货是基酒
- 需要存储5年以上才能出售
- 存得越久越值钱
- 所以周转慢是正常的！

---

## 存货周转率的评判标准

### 通用标准

| 周转率 | 周转天数 | 评价 |
|--------|----------|------|
| **> 10次/年** | < 37天 | 极快 ✅（生鲜、快消品） |
| **5-10次/年** | 37-73天 | 很快 ✅（零售） |
| **3-5次/年** | 73-122天 | 正常（制造业） |
| **1-3次/年** | 122-365天 | 较慢（耐用品） |
| **< 1次/年** | > 365天 | 很慢（茅台例外） |

---

### 行业差异

| 行业 | 典型周转率 | 原因 |
|------|-----------|------|
| **生鲜** | 20-50次/年 | 容易腐烂 |
| **快消品** | 10-20次/年 | 销售快 |
| **零售** | 5-10次/年 | 正常 |
| **制造业** | 3-5次/年 | 生产周期 |
| **白酒** | 0.2-0.5次/年 | 需要陈酿 |
| **房地产** | 0.1-0.3次/年 | 建设周期长 |

---

## 存货周转率的意义

### 1. 衡量销售能力

```
存货周转率高 = 卖得快
存货周转率低 = 卖得慢
```

| 公司A | 公司B |
|--------|--------|
| 存货周转率10次/年 | 存货周转率2次/年 |
| 每36天卖完存货 | 每182天卖完存货 |
| **销售能力强** ✅ | **销售能力弱** ❌ |

---

### 2. 衡量资金占用

```
存货周转率高 = 资金占用少
存货周转率低 = 资金占用多
```

| 公司A | 公司B |
|--------|--------|
| 年销售1000万 | 年销售1000万 |
| 存货周转率10次 | 存货周转率2次 |
| 平均存货100万 | 平均存货500万 |
| **资金占用少** ✅ | **资金占用多** ❌ |

**结论**：
- 公司A需要100万存货就够
- 公司B需要500万存货
- 公司A资金利用效率更高

---

### 3. 发现存货积压

```
存货周转率突然下降 = 可能存货积压
```

**例子**：某公司

| 年份 | 营业成本 | 存货 | 周转率 |
|------|----------|------|--------|
| 第1年 | 100亿 | 20亿 | 5次 ✅ |
| 第2年 | 100亿 | 40亿 | 2.5次 ⚠️ |
| 第3年 | 100亿 | 80亿 | 1.25次 ❌ |

**分析**：
- 营业成本不变
- 但存货逐年翻倍
- 周转率持续下降
- 说明**货卖不出去了**！

## 学习检查

**问题1**：A公司营业成本200亿，期末存货50亿。存货周转率是多少？周转天数是多少？

<details>
<summary>点击查看答案</summary>

存货周转率 = 200 ÷ 50 = 4次/年
存货周转天数 = 365 ÷ 4 = 91天

评价：正常水平，属于制造业的典型水平。

</details>

---

**问题2**：B公司营业成本100亿，存货周转率从5次下降到2次。这是什么信号？

<details>
<summary>点击查看答案</summary>

危险信号 ⚠️
- 存货周转率大幅下降
- 可能是存货积压
- 或者销售下滑
- 要深入分析原因

</details>

---

**问题3**：为什么茅台的存货周转率很低（0.18次/年），但依然是好公司？

<details>
<summary>点击查看答案</summary>

茅台的特殊性：
- 存货主要是基酒
- 需要存储5年以上
- 存得越久越值钱
- 周转慢是特色，不是问题
- 毛利率极高（91%），不需要靠周转率赚钱

</details>

## 快速记忆

```
存货周转率 = 营业成本 ÷ 平均存货
存货周转天数 = 365 ÷ 存货周转率

通用标准：
> 10次/年 ✅ → 极快（生鲜）
5-10次/年 → 很快（零售）
3-5次/年 → 正常（制造）
< 1次/年 → 很慢（茅台例外）

分析要点：
1. 和同行对比
2. 看趋势变化
3. 结合存货绝对值
4. 看行业特点

陷阱：
1. 降价促销提高周转率
2. 缺货导致周转率虚高
3. 周转率不是越高越好
```
