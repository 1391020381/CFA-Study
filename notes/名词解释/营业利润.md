# 营业利润（Operating Profit）

## 定义

**营业利润** = 公司**主营业务**产生的利润。

简单来说：**公司靠主业真正赚到的钱 —— 还没算投资收益、卖资产赚的钱、交税之前的利润。**

---

## 核心公式

```
营业利润 = 营业收入 - 营业成本 - 期间费用
         = 毛利 - 销售费用 - 管理费用 - 研发费用 - 财务费用
```

---

## 营业利润的意义

### 1. 衡量主业盈利能力

```
营业利润 > 0 → 主业赚钱
营业利润 < 0 → 主业亏钱
```

| 营业利润情况 | 含义 |
|--------------|------|
| 营业利润高 | 主业赚钱能力强 ✅ |
| 营业利润低 | 主业赚钱能力弱 |
| 营业利润为负 | 主业在亏本 ❌ |

**关键**：营业利润反映的是**主业**的盈利能力，不包括投资、卖资产等"副业"。

---

### 2. 排除"噪音"

营业利润**不包括**：
- 营业外收入（政府补贴、卖废品）
- 营业外支出（捐赠、罚款）
- 投资收益（分红、炒股）
- 所得税

**为什么要排除？**
- 这些收入不稳定、不可持续
- 不能反映公司真正的赚钱能力

---

## 营业利润的构成

### 利润表结构

```
营业收入 750亿
    │
    ├─ 减：营业成本 -70亿
    │
    ├─ = 毛利 680亿
    │
    ├─ 减：销售费用 -40亿
    ├─ 减：管理费用 -60亿
    ├─ 减：研发费用 -10亿
    ├─ 减：财务费用 +10亿（利息收入）
    │
    └─ = 营业利润 580亿 ✅
```

**茅台2023年（估算）**：
- 营业利润约**580亿元**
- 营业利润率约**77%**（580 ÷ 750）

---

## 营业利润率

### 营业利润率计算

```
营业利润率 = 营业利润 ÷ 营业收入 × 100%
```

| 营业利润率水平 | 评价 |
|---------------|------|
| > 30% | 极高 ✅（茅台77%） |
| 20%-30% | 很好 |
| 10%-20% | 良好 |
| 5%-10% | 一般 |
| < 5% | 很低 ⚠️ |
| < 0% | 主业亏损 ❌ |

---

### 行业营业利润率对比

| 行业 | 营业利润率 | 原因 |
|------|-----------|------|
| **白酒高端** | 50%-80% | 定价权强、费用率低 |
| **软件/SaaS** | 20%-40% | 边际成本低 |
| **奢侈品** | 20%-35% | 品牌溢价 |
| **医药** | 15%-30% | 研发费用高 |
| **零售** | 3%-8% | 竞争激烈 |
| **航空** | 5%-15% | 成本高、周期性 |
| **钢铁** | 3%-10% | 重资产、周期波动大 |

## 茅台的营业利润

### 茅台营业利润率（估算）

| 年份 | 收入（亿） | 营业利润（亿） | 营业利润率 |
|------|-----------|---------------|-----------|
| 2018 | 600 | 450 | 75% |
| 2019 | 650 | 500 | 77% |
| 2020 | 700 | 540 | 77% |
| 2021 | 750 | 580 | 77% |
| 2022 | 800 | 620 | 78% |
| 2023 | 750 | 580 | 77% |

**分析**：
- 营业利润率稳定在**75%-78%**
- 说明主业盈利能力极强且稳定
- 这是茅台的核心竞争力

---

### 为什么茅台营业利润率这么高？

| 原因 | 说明 |
|------|------|
| **毛利率极高** | 91%，行业最高 |
| **期间费用率极低** | 13%，远低于同行 |
| **财务费用为负** | 现金太多，利息收入 |
| **规模效应** | 收入增长，费用率下降 |

**公式**：
```
营业利润率 = 毛利率 - 期间费用率
           = 91% - 13%
           = 78%
```

## 营业利润 vs 其他利润概念

### 利润层次

| 利润类型 | 公式 | 意义 |
|----------|------|------|
| **毛利** | 收入 - 成本 | 产品本身赚多少 |
| **营业利润** | 毛利 - 期间费用 | 主业赚多少 |
| **利润总额** | 营业利润 + 营业外收支 | 全部赚多少 |
| **净利润** | 利润总额 - 所得税 | 最终赚多少 |

---

### 茅台的利润层次（2023年估算）

| 利润类型 | 金额（亿） | 利润率 |
|----------|-----------|--------|
| 毛利 | 680 | 91% |
| 营业利润 | 580 | 77% |
| 利润总额 | 580 | 77%（营业外收支很少） |
| 净利润 | 435 | 58%（所得税率约25%） |

**分析**：
- 从毛利91% → 净利润58%
- 中间只扣了33%（期间费用+所得税）
- 说明运营效率极高！

## 营业利润的"陷阱"

### 陷阱1：主业亏损，靠副业补

**例子**：某制造公司

| 项目 | 金额 |
|------|------|
| 营业利润 | -10亿（主业亏钱） |
| 投资收益 | +20亿（卖子公司） |
| 利润总额 | +10亿（看起来赚钱） |

**问题**：
- 主业在亏钱
- 靠卖资产维持盈利
- 不可持续 ❌

**判断**：重点看营业利润，不要被利润总额迷惑！

---

### 陷阱2：政府补贴掩盖亏损

**例子**：某新能源公司

| 项目 | 金额 |
|------|------|
| 营业利润 | -5亿（主业亏钱） |
| 政府补贴 | +8亿 |
| 利润总额 | +3亿（看起来赚钱） |

**问题**：
- 主业造血能力差
- 靠补贴生存
- 补贴减少就会亏损 ❌

---

### 陷阱3：营业利润虚高

**手段**：少计提费用

**例子**：
- 应该折旧的资产不折旧
- 应该计提的减值不计提
- 应该摊销的费用不摊销

**后果**：
- 当期营业利润虚高
- 未来需要补提，利润暴跌

---

## 如何用营业利润分析公司

### 第一步：看营业利润水平

```
营业利润率 > 20% = 优秀 ✅
营业利润率 10%-20% = 良好
营业利润率 < 10% = 需要警惕 ⚠️
营业利润率 < 0% = 主业亏损 ❌
```

---

### 第二步：看营业利润趋势

```
连续增长 = 主业盈利能力提升 ✅
连续下降 = 主业盈利能力恶化 ⚠️
波动大 = 行业周期性强
```

---

### 第三步：看营业利润 vs 净利润

| 情况 | 含义 |
|------|------|
| 营业利润 ≈ 净利润 | 盈利来自主业 ✅ |
| 净利润 >> 营业利润 | 盈利来自副业 ⚠️ |
| 净利润 << 营业利润 | 所得税或营业外支出大 ⚠️ |

**茅台**：营业利润 ≈ 净利润 + 所得税（正常）

---

### 第四步：看营业利润与同行对比

```
营业利润率 > 同行 = 竞争优势 ✅
营业利润率 < 同行 = 竞争劣势 ⚠️
```

## 学习检查

**问题1**：A公司营业利润率30%，B公司营业利润率5%。谁的赚钱能力更强？

<details>
<summary>点击查看答案</summary>

A公司赚钱能力更强。
- 营业利润率反映主业盈利能力
- A公司主业赚钱能力强
- B公司主业赚钱能力弱

</details>

---

**问题2**：C公司净利润10亿，但营业利润-5亿。这意味着什么？

<details>
<summary>点击查看答案</summary>

危险信号 ⚠️
- 主业在亏钱（营业利润-5亿）
- 靠副业或补贴维持盈利（净利润10亿）
- 不可持续，副业收入可能中断

</details>

---

**问题3**：为什么茅台的营业利润率高达77%？

<details>
<summary>点击查看答案</summary>

- 毛利率极高（91%）
- 期间费用率极低（13%）
- 营业利润率 = 91% - 13% = 78%
- 产品抢手，不需要花大量费用卖货

</details>
