# 留存收益（Retained Earnings）

## 定义

**留存收益** = 公司历年赚的净利润，**分给股东后剩下的部分**。

简单来说：**公司自我造血积累的财富 —— 赚了钱没分掉，留着继续发展的钱。**

---

## 核心公式

```
留存收益 = 累计净利润 - 累计分红
```

换个角度：
```
留存收益（期末）= 留存收益（期初） + 本年净利润 - 本年分红
```

---

## 留存收益的来源

### 公司赚钱后的选择

当公司赚了净利润，面临两个选择：

| 选择 | 说明 | 结果 |
|------|------|------|
| **分红** | 给股东发钱 | 股东拿到现金 |
| **留存** | 留在公司再投资 | 变成留存收益 |

**茅台例子**（2023年简化）：

| 项目 | 金额 |
|------|------|
| 本年净利润 | 700亿元 |
| 分给股东（分红） | 300亿元 |
| 留在公司 | 400亿元 → **进入留存收益** |

---

## 留存收益的构成

在会计上，留存收益分成两部分：

```
留存收益 = 盈余公积 + 未分配利润
```

| 部分 | 说明 | 能分红吗？ |
|------|------|-----------|
| **盈余公积** | 法律规定必须留存的 | ❌ 一般不能 |
| **未分配利润** | 可以自由支配的 | ✅ 可以分红 |

**中国的规定**：
- 每年要提净利润的**10%**作为盈余公积
- 直到盈余公积达到股本的**50%**为止

---

## 留存收益的意义

### 1. 公司的成长资金

```
留存收益 = 内部融资 = 不要利息的钱
```

| 资金来源 | 成本 | 优点 | 缺点 |
|----------|------|------|------|
| **留存收益** | 无 | 免费、灵活 | 股东可能不满（不分红） |
| **银行贷款** | 利息 | 快速 | 要还、要利息 |
| **发行新股** | 发行成本 | 永久资金 | 稀释股权 |

**茅台的特点**：
- 大量留存收益
- 不用借钱就能扩张
- 财务成本极低

---

### 2. 股东财富的增长

```
留存收益增加 → 所有者权益增加 → 每股净资产增加 → 股价上涨
```

**茅台例子**（简化）：

| 年份 | 每股净资产 | 股价（约） |
|------|-----------|-----------|
| 2018年 | 100元 | 600元 |
| 2023年 | 166元 | 1800元 |

- 每股净资产增长66%
- 股价增长200%
- 留存收益创造的价值远超成本！

---

### 3. 分红的来源

```
分红只能从"未分配利润"中分！
```

**重要规则**：
- ✅ 可以从当年的净利润分
- ✅ 可以从以前累积的未分配利润分
- ❌ 不能从股本分（这是股东投的）
- ❌ 不能从资本公积分（有例外）

---

## 巴菲特的最爱

### 巴菲特的投资哲学

> "公司每留存1元利润，至少要为股东创造1元的市场价值。"

**好公司（如茅台、苹果）**：
- 留存收益增长10亿
- 市值增长30亿 ✅
- 说明：公司善于用钱

**坏公司**：
- 留存收益增长10亿
- 市值不涨或下跌 ❌
- 说明：公司乱花钱

---

### 伯克希尔·哈撒韦的案例

巴菲特的伯克希尔公司：
- **从不分红**
- 所有利润都留存
- 用留存收益收购好公司
- 结果：股价从19美元涨到60万美元+

**为什么不分红？**
- 巴菲特能比股东更好地投资这笔钱
- 留存收益的回报率 > 股东自己投资的回报率

---

## 留存收益为负数？

### 什么是累计亏损？

```
留存收益 < 0 = 公司历年亏损超过盈利
```

**例子**：某创业公司

| 年份 | 净利润 | 分红 | 留存收益变动 |
|------|--------|------|--------------|
| 第1年 | -100万 | 0 | -100万 |
| 第2年 | -50万 | 0 | -50万 |
| 第3年 | +30万 | 0 | +30万 |
| **累计** | **-120万** | 0 | **-120万** |

**留存收益为负的后果**：
- 不能分红（法律规定）
- 可能面临退市风险
- 需要股东注资

---

## 股息支付率

### 多少利润应该分红？

```
股息支付率 = 分红 ÷ 净利润
```

| 公司类型 | 股息支付率 | 例子 |
|----------|-----------|------|
| **成熟公司** | 40%-60% | 茅台约50% |
| **成长公司** | 0%-20% | 亚马逊0% |
| **REITs** | 90%+ | 必须分红 |

**茅台的分红策略**（简化）：
- 净利润700亿
- 分红约350亿（支付率约50%）
- 留存约350亿（继续扩大生产）

**这是很健康的策略**：
- 股东拿到现金
- 公司也有钱发展

---

## 留存收益 vs 回购股票

### 两种方式回报股东

| 方式 | 说明 | 例子 |
|------|------|------|
| **分红** | 直接发现金 | 茅台每10股派200元 |
| **回购** | 买回股票注销 | 苹果经常回购 |

**回购的优势**：
- 股数减少
- 每股收益增加
- 股价上涨
- 有税收优势

**茅台的做法**：
- 既分红（每年几百亿）
- 也偶尔回购（较少）

---

## 财务比率中的应用

### ROE（净资产收益率）

```
ROE = 净利润 ÷ 所有者权益
```

其中，所有者权益主要是：
- 股本（通常很小）
- 留存收益（通常很大）

**茅台2023年（估算）**：
- 净利润：700亿
- 所有者权益：2090亿（主要是留存收益）
- ROE = 700 ÷ 2090 ≈ **33%** ✅ 很优秀！

---

## 警惕"虚假留存收益"

### 留存收益可能被"锁定"

| 情况 | 说明 |
|------|------|
| **对外担保** | 留存收益被用于担保 |
| **法律限制** | 某些行业有特殊限制 |
| **子公司亏损** | 母公司有留存收益，但子公司需要钱 |

**关键**：看现金流，不只是看留存收益！

## 学习检查

**问题1**：A公司第一年赚100万，分红40万；第二年赚150万，分红60万。第二年年末的留存收益是多少？

<details>
<summary>点击查看答案</summary>

- 第1年留存：100 - 40 = 60万
- 第2年留存：150 - 60 = 90万
- 第2年末留存收益：60 + 90 = 150万

</details>

---

**问题2**：为什么茅台把一半利润分红，一半留存，而不是全部分掉或全部留存？

<details>
<summary>点击查看答案</summary>

- 全部分掉：没钱扩大生产
- 全部留存：股东拿不到现金回报
- 一半一半：既有现金回报，又能发展 ✅

</details>

---

**问题3**：B公司留存收益10亿，但现金只有1亿，这可能吗？为什么？

<details>
<summary>点击查看答案</summary>

- 可能！
- 留存收益只是会计数字
- 现金可能已经用于：
  - 买固定资产（厂房、设备）
  - 买存货
  - 还债
- 所以：看公司要同时看"留存收益"和"现金"

</details>
