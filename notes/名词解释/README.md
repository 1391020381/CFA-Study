## 学习建议

1. **先掌握 TVM 基础**
   - 货币时间价值是所有计算的基础
   - 现值、终值、年金必须熟练

2. **理解经济指标的含义**
   - CPI、PMI、PPI、GDP 的关系
   - 如何解读这些数据

3. **结合实际应用**
   - 用自己的120w资产做练习
   - 思考这些概念如何影响投资决策

4. **不要死记公式**
   - CFP考试会提供公式表
   - 重要的是理解何时用哪个公式

## 名词关系图

```
货币时间价值 (TVM)
├─ 现值 (PV) ────────┐
├─ 终值 (FV) ────────┤
├─ 年金 ──────────────┤─ 相互转换计算
└─ 复利 ──────────────┘

经济指标体系
├─ 通胀衡量
│  ├─ CPI (消费品价格)
│  └─ PPI (工业品价格) → 领先 CPI
├─ 经济总量
│  └─ GDP
├─ 先行指标
│  └─ PMI → 领先 GDP
└─ 通胀调整
   └─ 名义利率 → 实际利率

经济周期
├─ 复苏 → PMI > 50, GDP ↑
├─ 扩张 → CPI ↑, 可能加息
├─ 衰退 → PMI < 50, GDP ↓
└─ 萧条 → 可能降息

投资估值指标
├─ 每股收益 (EPS) ────────────┐
│   └─ 净利润 ÷ 总股本         │
├─ 市盈率 (PE) ───────────────┤─ 股票估值核心
│   └─ 股价 ÷ EPS             │
└─ PEG 指标 ──────────────────┘
    └─ PE ÷ 盈利增长率

投资决策
├─ 机会成本 → 比较不同选择
├─ 实际利率 → 真实收益
├─ 经济周期 → 资产配置
└─ 估值指标 → 判断股价合理性
    ├─ PE → 基础估值
    ├─ PEG → 考虑成长性
    └─ EPS → 盈利能力

三大财务报表（FSA基础）
├─ 资产负债表（照片）
│   ├─ 资产 = 负债 + 所有者权益
│   └─ 特定时点快照
├─ 利润表（视频）
│   ├─ 收入 - 成本 - 费用 = 利润
│   └─ 一段时期业绩
└─ 现金流量表（流水账）
    ├─ 经营 + 投资 + 筹资活动
    └─ 真实现金进出

勾稽关系
└─ 利润表净利润 → 资产负债表留存收益
   └─ 现金流量表现金 → 资产负债表货币资金
```
