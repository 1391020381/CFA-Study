# 毛利率（Gross Profit Margin）

## 定义

**毛利率** = 毛利占营业收入的百分比。

简单来说：**每卖100块钱，扣除直接成本后，剩下多少块钱 —— 衡量产品本身赚钱能力的核心指标。**

---

## 核心公式

```
毛利率 = 毛利 ÷ 营业收入 × 100%
      = (营业收入 - 营业成本) ÷ 营业收入 × 100%
```

---

## 毛利率的意义

### 1. 衡量定价能力

```
毛利率高 = 可以定高价 = 定价权强
```

| 毛利率 | 定价能力 | 例子 |
|--------|----------|------|
| > 70% | 极强 ✅ | 茅台91%、软件80%+ |
| 50%-70% | 很强 | 苹果40%+、奢侈品 |
| 30%-50% | 一般 | 制造业 |
| < 30% | 较弱 | 零售、加工 |

---

### 2. 衡量成本控制能力

```
毛利率 = 1 - 成本收入比
```

| 成本控制 | 毛利率表现 |
|----------|-----------|
| 优秀 | 毛利率持续上升或稳定在高水平 |
| 一般 | 毛利率中等 |
| 较差 | 毛利率下降或很低 |

---

### 3. 判断竞争优势

```
毛利率 > 同行 = 竞争优势
```

**白酒行业对比**（2023年估算）：

| 公司 | 毛利率 | 竞争优势 |
|------|--------|----------|
| **茅台** | 91% | 极强，行业龙头 |
| 五粮液 | 75% | 很强，行业老二 |
| 泸州老窖 | 70% | 强 |
| 洋河股份 | 65% | 较强 |
| 低端白酒 | 40%-60% | 一般 |

**结论**：茅台竞争优势明显！

---

## 茅台的毛利率分析

### 茅台毛利率历史（估算）

| 年份 | 收入（亿） | 成本（亿） | 毛利率 |
|------|-----------|-----------|--------|
| 2018 | 600 | 60 | 90% |
| 2019 | 650 | 65 | 90% |
| 2020 | 700 | 63 | 91% |
| 2021 | 750 | 68 | 91% |
| 2022 | 800 | 72 | 91% |
| 2023 | 750 | 68 | 91% |

**分析**：
- 毛利率稳定在90%-91%
- 说明定价权稳定
- 成本控制优秀

---

### 为什么茅台毛利率这么高？

| 原因 | 说明 |
|------|------|
| **品牌力** | 茅台=面子，社交必需品 |
| **稀缺性** | 只有茅台镇能产，产能有限 |
| **定价权** | 可以不断提价，需求不减 |
| **成本低** | 粮食成本占收入比例很低 |
| **供不应求** | 经销商预付排队拿货 |

|------|-----------|------|

| **钢铁** | 5%-15% | 重资产、周期 |
| **建筑** | 5%-15% | 竞争激烈 |

## 毛利率的变化分析

### 毛利率上升（好事）

| 原因 | 例子 |
|------|------|
| 产品提价 | 茅台提价 |
| 成本下降 | 原材料降价 |
| 产品结构优化 | 多卖高毛利产品 |
| 规模效应 | 产量增加，单位成本下降 |

---

### 毛利率下降（坏事）

| 原因 | 例子 |
|------|------|
| 价格战 | 各大车企降价 |
| 成本上升 | 猪肉涨价，肉制品毛利率下降 |
| 产品结构恶化 | 多卖低毛利产品 |
| 竞争加剧 | 新进入者抢市场 |

---

## 毛利率与净利润率

### 从毛利率到净利润率

```
毛利率 - 期间费用率 = 营业利润率
营业利润率 - 所得税率 = 净利润率
```

**茅台例子**（2023年估算）：

| 指标 | 数值 | 说明 |
|------|------|------|
| 毛利率 | 91% | 极高 |
| 销售费用率 | 5% | 很低 |
| 管理费用率 | 8% | 适中 |
| 研发费用率 | 1% | 很低 |
| 财务费用率 | -1% | 利息收入 |
| 营业利润率 | 约70% | 极高 |
| 净利润率 | 约50% | 极高 |

**对比**：

| 公司 | 毛利率 | 净利润率 | 差异 |
|------|--------|----------|------|
| **茅台** | 91% | 50% | 41% |
| **普通制造** | 30% | 5% | 25% |

**茅台优势**：期间费用率很低，净利率极高！

## 毛利率的"陷阱"

### 陷阱1：虚假高毛利率

**手段**：少结转成本

**识别方法**：
| 方法 | 说明 |
|------|------|
| 对比同行 | 毛利率显著高于同行 |
| 检查存货 | 存货周转率异常下降 |
| 检查现金流 | 经营现金流与净利润不匹配 |

---

### 陷阱2：不可持续的高毛利率

**原因**：一次性因素

| 因素 | 例子 |
|------|------|
| 原材料价格短期下降 | 2020年原油价格暴跌，石化毛利率上升 |
| 短缺带来的涨价 | 疫情期间，口罩毛利率暴涨 |

**判断**：看长期毛利率是否稳定

---

### 陷阱3："高毛利率"但"低净利率"

**原因**：期间费用过高

**例子**：某网红品牌

| 项目 | 数值 |
|------|------|
| 毛利率 | 70%（很好） |
| 销售费用率 | 50%（疯狂打广告） |
| 净利润率 | 5%（很差） |

**问题**：过度依赖营销，不可持续

## 如何用毛利率选股

### 第一步：看毛利率水平

```
毛利率 > 50% = 进入候选名单
```

---

### 第二步：看毛利率趋势

```
连续3-5年稳定或上升 = 通过 ✅
```

---

### 第三步：看毛利率与同行对比

```
毛利率 > 同行平均 = 竞争优势 ✅
```

---

### 第四步：看毛利率与净利率差额

```
差额小 = 期间费用控制好 ✅
```

## 学习检查

**问题1**：A公司毛利率40%，B公司毛利率60%。谁的定价权更强？

<details>
<summary>点击查看答案</summary>

B公司定价权更强。
- 毛利率高 = 可以定高价
- 说明产品竞争力更强
- 品牌或技术优势更明显

</details>

---

**问题2**：茅台毛利率91%，净利率50%。中间差了41%，去哪了？

<details>
<summary>点击查看答案</summary>

扣除期间费用和所得税：
- 销售费用（5%）
- 管理费用（8%）
- 研发费用（1%）
- 财务费用（-1%，即收入）
- 所得税（约25%）
- 合计约41%

</details>

---

**问题3**：某公司毛利率突然从30%跳到50%，但行业平均只有25%。你应该警惕什么？

<details>
<summary>点击查看答案</summary>

- 可能是财务造假（少结转成本）
- 毛利率显著高于同行不正常
- 要检查现金流是否匹配
- 要看存货是否异常

</details>

## 快速记忆

```
毛利率 = (收入 - 成本) ÷ 收入

> 70% → 极强（茅台、软件）
50%-70% → 很好
30%-50% → 一般
< 30% → 较弱

判断标准：
1. 绝对水平（越高越好）
2. 历史趋势（稳定或上升）
3. 同行对比（高于平均）
4. 可持续性（长期稳定）

警惕：
1. 突然变化
2. 显著高于同行
3. 高毛利低净利
```
