## 提出的问题与回答

### 问题 1: 利润表的收入来源
**问题**: 茅台是怎么赚钱的？利润表上会有哪些具体的收入项目？

**学生的回答**: 预付款、现金、理财（混淆了资产和收入）

**纠正**: 区分了资产负债表的"资产"和利润表的"收入"
- 资产 = 公司拥有的东西
- 收入 = 公司赚的钱

**讲解内容**:
- 营业收入 = 卖货收到的钱（茅台：卖酒）
- 营业成本 = 生产产品直接花的钱
- 毛利 = 收入 - 成本
- 期间费用 = 销售费用、管理费用、研发费用、财务费用
- 营业利润 = 毛利 - 期间费用
- 净利润 = 最终属于股东的利润

**理解检查**: ✅ 学生理解了利润表的完整结构

### 问题 2: 现金流量表的三大类活动
**问题**: 现金流量表的三大类活动是什么？

**学生的回答**: 记得不清楚

**讲解内容**:
- 经营活动 = 做生意（卖货、买原料、发工资）
- 投资活动 = 长期投资（建厂房、买设备、买理财）
- 筹资活动 = 找钱（借钱、分红、回购股票）

**练习**: 判断以下活动类型
1. 卖茅台酒收到经销商付款 → 经营活动 ✅
2. 给股东分红 → 筹资活动 ✅
3. 建新酿酒车间 → 投资活动 ✅
4. 发工人工资 → 经营活动 ✅
5. 找银行借了100万 → 筹资活动 ✅

**理解检查**: ✅ 全部回答正确

### 问题 3: 财务比率的理解
**问题**: 你对 ROE 和资产负债率了解多少？

**学生的回答**:
- ROE: 用来衡量赚钱的速度
- 资产负债率: 越低越好

**补充讲解**:

**ROE（净资产收益率）**:
- 更准确的定义：股东每投100元，公司能赚多少
- 公式：ROE = 净利润 ÷ 所有者权益
- 茅台ROE约19%（优秀）
- 巴菲特最看重的指标

**资产负债率**:
- 公式：总负债 ÷ 总资产
- 茅台约18%（极低，风险低）
- 要和同行对比（银行90%+是正常的）

**理解检查**: ✅ 学生理解了这两个指标

### 问题 4: 财务比率计算练习
**问题**: A公司总资产100亿，总负债30亿，净利润15亿，所有者权益70亿。

**学生的回答**:
- 资产负债率 = 30/100
- ROE = 15/70
- 评价：负债在可控范围内

**确认**: 完全正确！✅
- 资产负债率 = 30%（健康）
- ROE ≈ 21.4%（优秀）
- 综合评价：财务健康且盈利能力强的好公司
